## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失部分位于多面体引力势的计算中，需要补全面和边的张量积近似贡献。对于面贡献：先调用已有函数计算面立体角，求取面的重心位置向量；根据引力常数、密度、面重心向量与单位法向量点乘的平方，乘以负零点五的系数以及面立体角，计算面势并累加。对于边贡献：遍历当前面的三条边，获取边两端顶点索引并计算相对向量，调用已有函数计算边因子；接着计算边向量及其长度平方，若长度平方小于极小阈值（如1e-14）则跳过；构造边张量矩阵（由两端相对向量的外积之和构成），基于引力常数、密度、相对向量与该张量矩阵的点乘结果、边因子及边长平方，乘以零点五的系数计算边势并累加。

第二个缺失部分位于组合引力加速度的计算流程中，需要补全单位转换与模型混合逻辑。首先调用已有的多面体引力加速度计算函数，并将结果乘以千分之一以统一为千米每秒平方的单位。接着实现平滑过渡权重：设定过渡带宽为参考半径的一半，利用双曲正切函数基于当前位置距离与过渡边界的差值计算初始权重，再将权重截断在0到1的闭区间内。最后，利用该权重对多面体加速度和球谐加速度进行线性插值组合并返回。

第三个缺失部分位于引力模型的测试循环中，需要补全遍历计算与结果打印逻辑。对于每个测试位置，计算其向量的二范数作为距离。调用组合引力模型函数，传入位置、网格、千米制的引力参数与参考半径、球谐系数，并设定最大截断阶数为6、过渡半径倍数为2.5等参数；同时调用球谐模型的加速度方法获取纯球谐结果。最后，格式化输出该位置坐标及距离（保留两位小数），以及组合加速度和纯球谐加速度的大小（以科学计数法保留六位小数）。
