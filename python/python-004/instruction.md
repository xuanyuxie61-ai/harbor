本项目涉及引力波信号处理与准正规模（QNM）的分析。缺失的代码主要分为三个部分：

第一部分：实现一个从已知根构造多项式的功能。算法思路是利用根与多项式的关系，将多项式系数初始为单位1，然后遍历复数根数组，通过连续卷积操作逐步将每个根对应的一次项乘入多项式中，最终得到完整的多项式系数数组，并连同根数组一并返回。

第二部分：实现引力波环形下降信号的叠加计算。算法思路是遍历由角量子数、磁量子数和过冲数标记的准正规模频率字典。对于每个模式，获取其对应振幅（若字典中无对应项则按默认规则赋值，默认值与角量子数加一后的倒数相关）。信号仅在时间大于或等于合并时间时激发。此处需要实现复指数衰减方程来描述每个模式随时间演化的衰减振荡，并将所有模式的贡献累加到波形变量中。

第三部分：调用已有的准正规模频率求解函数，并输出相关信息。调用时需传入最终质量和自旋参数，并将最大角量子数设为3、过冲数设为1。首先打印求解得到的频率数量；其次，检查频率字典中是否存在角量子数为2、磁量子数为2、过冲数为0的特定模式，若存在，则分别打印该模式频率的实部和虚部，均保留六位小数。
