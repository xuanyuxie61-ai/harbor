# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失的类方法用于提取今天的转移函数。该函数需要先调用求解方法获取相关物理量，然后寻找复合时期（即光学深度累积值约为1.0的时刻）。接着，利用声视界的近似值（约150.0 Mpc）计算参数。当参数极小（小于1e-6）时，需直接返回1.0。在Sachs-Wolfe近似下，需实现结合声学振荡的转移函数计算逻辑，其中复合时期的引力势需根据超视界守恒近似进行估算（即与光子单体微扰成负三分之一比例）。最后，引入Silk阻尼效应，使用特定的阻尼波数（0.14 Mpc的负一次方）对结果进行指数衰减处理。

第二个缺失的类方法是参数化转移函数的解析表达式。该函数需先处理波数非正的边界情况。对于正常情况，计算球Bessel函数的参数（由波数乘以当前时间与复合时期的时间差得到），并求取对应阶数的球Bessel函数值。此外，还需引入Silk阻尼效应（同样基于阻尼波数的指数衰减），以及声学峰的调制项（通过包含特定峰位波数的正弦函数实现），最后将这三部分相乘返回。

第三个缺失的独立函数用于通过视线积分计算功率谱单极矩。该函数需使用指定的快速数值积分方法进行积分。被积函数需处理非正波数的边界情况，对于正常情况，计算原初功率谱与转移函数平方的乘积，并除以波数。最终，将积分结果乘以一个特定的常数系数（4π）作为输出。
