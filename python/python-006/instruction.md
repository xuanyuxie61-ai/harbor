## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

这是一个关于天体物理结构计算的项目，涉及TOV方程的数值求解和物态方程的计算。项目中有以下三处代码缺失：

1. 缺失了一个类方法，用于执行常微分方程数值积分的单步推进。该方法需实现经典的四阶龙格-库塔（RK4）算法思路：计算四个中间斜率（这些斜率通过调用项目中已正确实现的TOV方程函数获得，并需传入物态方程相关的参数），然后将这四个斜率按1、2、2、1的权重比例组合，以更新并返回当前的状态向量。

2. 缺失了一段计算压强的代码，该部分需分别计算动能压强与势能压强并返回两者之和。其中，动能压强的计算涉及约化普朗克常数与质量之比、费米动量、重子数密度，以及同位旋不对称度的五分之三次幂的特定线性组合；势能压强包含两体和三体相互作用项，涉及参数t0、x0、t3、x3及alpha，计算时需考虑重子数密度的幂次（两体项为平方，三体项为alpha加2的幂）以及不对称度的平方项。

3. 缺失了一行用于构建物态方程中能量密度与压强函数关系的代码。需调用项目中已正确实现的简化构建函数，并传入多方指数（Gamma）参数，其值应设为2.5。
