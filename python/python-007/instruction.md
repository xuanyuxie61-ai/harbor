## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

项目涉及天体物理吸积盘的数值模拟。缺失的代码分为三个部分：

第一部分位于一个计算吸积盘稳态参数的函数末尾，需要根据辐射主导的吸积盘模型依次推导并返回表面密度、温度和标高。首先，需要实现标准薄盘的辐射主导温度方程，计算时需引入一个与最内稳定圆轨道半径相关的修正因子，该因子的值应被截断在极小正值和1之间；计算出的温度需确保下限为1。接着，调用已有函数计算标高和声速，两者均需设置极小正值作为下限保护。然后，根据alpha粘滞律计算运动粘度，同样赋予极小正值下限。最后，基于质量守恒与粘滞定律计算表面密度并乘以前述修正因子，需确保其值非负。

第二部分位于主流程中，用于获取吸积盘在空间网格上的物理量分布。需依次调用已有的五个函数：先调用获取标准盘参数的函数得到表面密度、盘温度和盘标高；随后调用计算径向速度、开普勒角速度的函数；最后调用计算声速的函数，并传入盘温度参数。

第三部分位于求解偏微分方程的矩阵构建逻辑中，用于为离散化后的线性方程组施加左右边界条件。对于Dirichlet（第一类）边界条件，需将系数矩阵对应边界行的所有元素置零，对角线元素置一，并将右端向量在该行的值设为给定的边界值。对于Neumann（第二类）边界条件，需将给定的梯度值累加到右端向量对应行的元素上。左右边界需分别独立判断并处理。
