## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个宇宙学N体模拟相关的计算代码，包含以下几个核心缺失部分：

1. 线性增长因子微分方程右端项计算函数：该函数用于计算线性增长因子关于尺度因子的二阶常微分方程转化为一阶系统后的右端项。算法思路是将二阶方程降阶为一阶方程组，提取状态变量及其导数，并计算它们的随时间变化率。需要注意，当尺度因子小于或等于0时，需将其赋为一个极小的正数（如1e-10）以避免除零错误。计算中涉及两个关键系数，分别与哈勃参数的对数导数以及物质密度参数有关。此处需要实现线性增长因子的二阶微分方程，方程中包含哈勃参数对数导数项和物质密度参数项。函数中调用的哈勃参数对数导数计算方法和物质密度参数计算方法默认已正确实现。

2. 泊松方程FFT求解函数：该函数用于通过快速傅里叶变换求解泊松方程以获取引力势。算法思路是首先将实空间的密度对比度场转换到傅里叶空间，在频域中利用预先定义的格林函数与尺度因子的平方、变换后的密度场进行乘积运算，最后逆变换回实空间并提取实部。在此过程中，正向傅里叶变换和逆向傅里叶变换需要使用网格总数的立方进行归一化处理。此处需要实现傅里叶空间下的泊松方程求解逻辑，频域引力势与尺度因子平方、密度场傅里叶系数及格林函数相关。类中已有的格林函数属性默认已正确初始化。

3. Leapfrog积分单步推进函数：该函数用于实现N体模拟中的Leapfrog（蛙跳）积分器的单步推进，采用Kick-Drift-Kick（KDK）模式。算法思路是先对速度进行半步更新，然后利用半步更新后的速度对位置进行全步更新，在新的位置上重新计算加速度，最后利用新加速度完成速度的剩余半步更新。在半步更新时，时间步长需取为输入步长的一半。此处需要实现KDK模式的蛙跳积分逻辑。类中用于更新速度和位置的步进函数，以及外部传入的加速度计算函数默认已正确实现。
