# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失的代码片段用于计算二维方格紧束缚模型的色散关系。此处需要实现二维方格紧束缚色散关系方程，该方程包含最近邻跃迁项、次近邻跃迁项以及化学势的偏移。在实现时，需要接收波矢的x和y分量作为输入，并包含三个具有默认值的参数：最近邻跃迁参数（默认值为1.0）、次近邻跃迁参数（默认值为0.3）和化学势（默认值为0.0）。

第二个缺失的代码片段用于计算超导能隙方程的被积函数。此处需要实现能隙方程的被积函数，它涉及d波形式因子、准粒子能量和双曲正切项的组合计算。具体而言，需要从输入的k点数组中提取波矢坐标，调用项目中已有的`d_wave_form_factor`和`quasiparticle_energy`函数分别计算形式因子和准粒子能量，并结合相互作用强度与逆温度等参数计算最终结果。特别需要注意的是，为了避免分母为零导致数值发散，需要对准粒子能量设置一个极小的下限进行截断。该函数的参数包括k点数组、能隙常数、相互作用强度、逆温度以及上述的跃迁和化学势参数。

第三个缺失的代码片段用于在整个二维布里渊区上执行数值积分。此处需要实现基于张量积的二维Gauss-Legendre求积算法。算法思路是：首先获取一维Gauss-Legendre节点和权重，然后将节点和权重映射到布里渊区范围，其中权重需要乘以相应的雅可比因子；接着利用网格构造二维节点坐标和对应的权重组合，最后对被积函数在所有节点上的值进行加权求和并返回结果。该函数接受一个被积函数和每维的Gauss点数（默认值为40）作为输入，积分区间为负π到正π的二维正方形区域。
