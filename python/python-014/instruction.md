## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失片段的功能是实现使用单位四元数旋转三维矢量的函数。算法思路上，此处需要实现四元数旋转变换方程，即将三维矢量转换为实部为零的纯四元数，与旋转四元数及其共轭进行特定的乘法操作。具体来说，首先需要将输入四元数归一化，然后将待旋转矢量构建为纯四元数形式，接着求出归一化四元数的共轭，最后依次进行两次四元数乘法，提取结果四元数的虚部即为旋转后的三维矢量。其中四元数的归一化、求共轭和乘法等基础操作已默认正确实现，可直接调用。

第二个缺失片段的功能是计算磁性自旋系统在特定旋转角度下的系统总能量。算法思路上，此处需要实现包含交换相互作用和单轴各向异性的能量计算方程。具体而言，首先需要将绕固定轴的旋转角度转换为四元数，并调用四元数旋转函数更新指定位置的自旋矢量，将其替换进原自旋数组得到新构型。接着计算交换能量：将耦合矩阵与新自旋构型进行矩阵乘法，再与新自旋逐元素相乘并求和，结果乘以0.5。然后计算各向异性能：使用0.05的系数乘以新自旋数组中所有Z方向分量的平方和。最后将两者相加返回总能量。此片段依赖的外部自旋数组、轴向量、位置索引和耦合矩阵均已在项目其他地方定义。

第三个缺失片段的功能是在一维有限元分析中，利用高斯积分循环计算单元的刚度矩阵与载荷向量并完成全局组装。算法思路上，此处需要实现一维变系数二阶微分方程的伽辽金弱形式积分，涵盖扩散项、反应项和源项的离散。具体而言，需要对两个高斯积分点进行遍历，在每个积分点处获取当前坐标下的变系数值，即扩散系数、反应系数和源项。随后分别计算左节点与右节点对应的基函数及其导数值。根据有限元组装规则，将单元长度与积分权重作为缩放因子，把扩散项和反应项的乘积累加到矩阵对角线位置（左节点对自身、右节点对自身），将源项与基函数的乘积累加到右端载荷向量的对应位置。同时计算左右节点基函数的交叉项并累加到矩阵的非对角线位置。涉及的基函数及其导数计算函数、变系数函数以及积分点坐标、权重等均已默认存在。
