## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及低维材料的紧束缚模型计算与电子半经典输运模拟。缺失的代码包含三个核心部分：

第一部分：计算紧束缚模型中的跃迁积分。该函数需要根据原子间的距离和相对角度计算跃迁参数。当距离小于极小阈值（1e-6）或大于3倍碳碳键长时，直接返回0.0作为平滑截断。对于有效距离，跃迁参数的值由三部分相乘决定：初始层内跃迁常数、随距离呈指数衰减的项（以碳碳键长为参考点），以及与角度相关的各向异性项（由极角的余弦平方项与正弦平方项乘以各向异性参数组合而成）。最终将结果转为浮点数返回。

第二部分：构建k空间哈密顿量并计算能带。需要遍历所有k点，初始化复数形式的哈密顿量矩阵。对于矩阵中的每个元素，利用实空间哈密顿量矩阵元乘以由Bloch定理引入的相位因子（该因子由k点与原子二维坐标差的点积计算得到）。构建完成后，需对矩阵进行厄米化处理（取原矩阵与其共轭转置的平均），随后调用已有的对角化函数求解本征值，并将结果存入能量数组中。

第三部分：计算电子在电磁场中的半经典运动状态导数。首先需要通过有限差分法计算群速度，差分步长设置为1e-4。在提取指定能带的能量值后，利用能量对k的梯度与约化普朗克常数的关系计算群速度。接着，根据半经典运动方程计算波矢对时间的导数，该导数包含电场力项和磁场洛伦兹力项（对于沿z轴的磁场，叉乘项需转化为二维速度分量的特定组合形式，磁场有效值在绝对值大于极小阈值时参与计算）。位置对时间的导数即为群速度。最后，将波矢和位置的导数拼接成一个数组返回。
