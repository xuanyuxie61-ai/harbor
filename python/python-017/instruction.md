# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个基于相场法的多铁性材料动力学模拟程序，涉及极化场（P）与磁化场（M）的耦合演化，核心物理模型基于含时金兹堡-朗道方程。

缺失部分包含三个功能模块，需要补全如下：

1. **计算自由能泛函对极化场P的变分导数**：此处需要实现朗道-金兹堡自由能模型中关于P的变分导数计算。首先提取P和M的各个分量，并计算P的模方以及P与M的叉积分量。接着，分别计算x和y方向的偏导数，这些导数由P的线性项、高阶非线性项、P与M的交叉耦合项以及包含拉普拉斯算子的梯度能项组成，各项系数由传入的参数对象决定。最终将两个方向的导数组合成数组返回。

2. **计算极化场P的反应项**：此处需要实现含时金兹堡-朗道方程中的反应项计算。初始化一个与P同形状的零数组用于存储结果，然后在空间内部网格上遍历（跳过边界）。在循环中，提取当前点的P和M构建局部向量（注意此处进行了简化，局部P仅包含x分量，局部M仅包含y分量），调用已有的函数计算全场的拉普拉斯张量并提取当前位置的值，再调用前述的变分导数计算函数。最后，将变分导数的x分量乘以负的动力学系数赋值给结果数组的对应位置，循环结束后返回结果数组。

3. **时间步进与场演化**：此处需要实现极化场和磁化场的时间推进。分别调用已有的P和M的求解器进行步进，求解器需接收当前场以及对应的反应项函数。注意在传入反应项函数时需使用匿名函数形式，且要体现两场的耦合关系：P的反应项函数依赖于当前演化的P和固定的M，而M的反应项函数依赖于固定的P和当前演化的M。最终返回更新后的两个场。
