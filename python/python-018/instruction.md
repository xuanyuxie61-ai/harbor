## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一维Kitaev链的拓扑性质分析，包括马约拉纳零能模的识别、拓扑不变量的计算以及相关物理量的格式化输出。项目中有三处代码缺失，具体功能与算法思路如下：

缺失部分1：位于零能模识别方法内部。该部分需要从计算出的零能特征向量中提取粒子分量和空穴分量。算法思路为：遍历零能索引，将特征向量拆分为前后两半部分，并验证马约拉纳条件（即粒子与空穴分量需有显著重叠）。计算重叠度时需进行归一化处理，并加入一个极小值以防除零。若重叠度大于设定的阈值（如0.5，表示放宽容差只需显著重叠即可），则保留该模式。最后需将保留的模式转化为数组返回，若无满足条件的模式则返回空数组。

缺失部分2：位于绕数计算方法内部。该部分需要实现一维Kitaev链的Z₂绕数拓扑不变量的计算。算法思路为：根据超导能隙的符号、化学势与跃迁幅度的组合符号，通过符号函数的乘积与差值计算得到绕数。具体而言，需先判断超导序参量的正负号（为零时取0），再分别判断化学势加上两倍跃迁幅度、以及化学势减去两倍跃迁幅度的正负号，最后通过这三个符号的组合进行整除运算得出整数型的拓扑不变量并返回。

缺失部分3：位于主程序或物理量分析流程中。该部分负责整合计算结果并进行格式化输出。首先需要调用求解器的零能模识别方法（需传入特征值、特征向量及设定的极小能量容差），获取马约拉纳零能模的两个分量。随后输出检测到的模数量；若存在零能模，需计算并输出第一个模边界波函数的最大绝对值，以及反参与率（IPR，用于衡量波函数局域化程度，越接近1越局域）。接着，遍历给定的化学势列表（包含从负3.0到正3.0间的若干典型值），调用绕数计算方法，输出对应的绕数值并判断其属于拓扑非平庸还是拓扑平庸。最后，针对另一组特定的化学势列表，调用纠缠熵计算方法（设定子系统大小为10），输出纠缠熵值，并同时输出绕数与理论值（自然对数的2的一半）的对比结果。
