你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个二维电子气在磁场下的Hartree-Fock自洽场计算。缺失的代码分布在四个位置，具体功能与算法思路如下：

第一处缺失：需要实现计算朗道轨道波函数的核心逻辑。此处需要实现朗道能级波函数公式，具体算法为：首先将输入转换为复数数组，并计算无量纲半径平方；接着利用已有的连带拉盖尔多项式计算函数求多项式部分的值；然后根据角量子数的正负情况分别计算归一化常数的平方，若角量子数为负且主量子数与其相加小于零，则返回与输入同形状的全零复数数组；之后计算包含符号、圆周率、磁长度及幂次的前置归一化系数；再分别计算波函数的角动量部分（复数网格与磁长度相关的幂次）和高斯包络衰减部分；最后将归一化系数、角动量部分、多项式值和高斯包络相乘得到波函数并返回。

第二处缺失：需要构建初始的基函数轨道。具体算法为：创建一个全零的复数三维数组用于存储基函数，将实部与虚部网格组合成复数网格；然后遍历基函数索引，按照对3取模得到主量子数、整除3得到角量子数的规则，调用已有的波函数计算函数填充该数组。

第三处缺失：需要构建Hartree-Fock矩阵并返回结果。此处分为三个步骤：首先计算Hartree势，通过双重循环遍历网格，计算各点间的距离，调用已有的二维库仑相互作用函数计算势能，并将势能与电子密度的乘积进行求和及离散面积积分得到Hartree势场；接着初始化全零的复数HF矩阵，在对角元上加上单粒子项，此过程需调用已有的朗道能级能量函数，并传入索引整除2的结果以及固定常数10.0和1.0；然后在对角元上累加Hartree势的贡献，即波函数共轭、Hartree势与波函数乘积的积分；最后处理非对角元，计算基函数间的重叠积分，乘以微小系数0.01作为近似值，并依据共轭关系保证矩阵的厄米性。完成后返回HF矩阵和电子密度。

第四处缺失：需要设置计算网格，调用自洽场求解函数并打印结果。具体算法为：在磁长度两倍的负值到正值范围内，根据网格分辨率生成一维坐标数组，并构建二维网格；调用已有的自洽场函数，传入电子数2、基函数数4、磁场强度、磁长度、二维网格以及最大迭代次数8和收敛容差1e-4；最后打印自洽场结果，包括收敛状态、前四个能级的实部、电子密度的最大值，以及通过离散积分计算出的总电子数。
