本项目是一个关于D-T聚变等离子体物理的数值模拟程序，主要用于计算聚变反应中各粒子数密度随时间的演化并分析稳态性能。

缺失的代码主要分为三个部分，对应的功能与算法思路如下：

1. 第一部分位于粒子数密度演化微分方程组的计算函数末尾。该函数需要实现D-T聚变反应中的粒子平衡模型，分别计算氘、氚、氦-4（灰）以及中子数密度随时间的变化率。缺失处需要将这四个变化率按顺序组装成一个一维数组并返回，注意需指定数组的数据类型为浮点型。

3. 第三部分位于稳态结果的输出与后续物理量计算区域。首先，需要将演化至稳态（即时间序列最后一组数据）时的氘、氚、氦-4数密度，以及之前已计算好的聚变功率密度和增益因子Q进行格式化打印输出。随后，需要计算电子密度以用于评估轫致辐射损失。此处需要基于等离子体准中性近似原理来推导电子密度，即电子密度由燃料离子和灰离子的密度与各自电荷数的乘积之和决定（氘电荷数为1，氦电荷数为2），请利用稳态的氘和氦-4密度计算出电子密度。
