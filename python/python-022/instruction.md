## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

项目中缺失了与计算物理量“电子数密度”相关的函数定义及调用逻辑，具体包含三个部分：

1. 需要定义一个计算电子数密度的通用函数。该函数接收密度、有效电荷和平均原子质量作为输入，需实现基于有效电荷、密度和阿伏伽德罗常数的物理方程来求解电子数密度。在计算时，分母中的平均原子质量需乘以千分之一进行单位换算。算法上需注意处理平均原子质量非正值的异常情况以避免除零错误，并要求支持数组形式的输入与运算。

2. 在某处计算逻辑中，需要对所有网格单元计算电子数密度。此时平均原子质量取固定的数值2.5，同样需要经过千分之一的单位换算。可以通过循环遍历每个网格单元来实现该计算过程。

3. 在另一处调用逻辑中，需要调用前面定义的通用函数来计算电子数密度。此时平均原子质量参数不再是固定值，而是需要根据网格单元的中心位置进行条件判断：如果位置大于等于特定的冰层半径阈值，则使用烧蚀层的平均原子质量参数；否则，平均原子质量使用固定值2.5。
