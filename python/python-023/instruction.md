# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一部分代码需要实现平行传播R模式（右旋波）的动力学色散函数及其对频率导数的计算。首先需要从参数字典中提取电子相关的物理常数（如电荷、质量、光速、真空介电常数、背景磁场、数密度等，以及由它们导出的特征频率和热速度）。为避免数值计算发散，需对波数和频率进行极小值边界检查。接着计算电子回旋共振参数，并调用已实现的等离子体色散函数及其一阶导数。在此基础上，按照R模式的动力学色散方程计算色散函数的值，并利用解析求导法则（注意等离子体色散函数的二阶导数可通过其一阶导数和原函数的线性组合表示）推导并计算该色散函数对频率的导数，最后返回这两个计算结果。

第二部分代码需要实现基于波粒相互作用的准线性扩散系数（平行、垂直及交叉分量）计算。需要从参数字典获取基本物理量，并在给定的平行和垂直速度网格上构建二维速度空间，计算洛伦兹因子并进行截断保护以避免数值异常。初始化三个扩散系数矩阵，并设定一个简化的波振幅常数。随后对已知的每一支波模式进行遍历：提取平行波数和复频率，过滤掉实部非正的频率。对于每支波，计算共振宽化宽度，并按固定比例估算垂直波数，进而计算贝塞尔函数参数及特定阶数的贝塞尔函数值。根据多普勒频移回旋共振条件计算共振函数，并用高斯函数近似宽化的狄拉克δ函数。结合物理常数和波振幅计算前置系数，并计算包含平行与垂直速度分量的投影因子（需进行截断保护）。将各波模式的贡献累加到三个扩散系数矩阵中，最后对结果施加物理约束（确保平行和垂直分量非负，交叉分量满足基于柯西-施瓦茨不等式的界限约束）。

第三部分代码用于判断粒子在速度空间中是否处于某一波模式的共振区域内。首先需要根据波的实部频率、电子回旋频率、洛伦兹因子和波数计算共振速度，并求出当前平行速度与共振速度的差值。为了综合平行和垂直方向速度偏差的影响，使用p-范数来度量速度空间中的距离：需要根据输入的范数类型参数（如2、1、无穷大或一般p值），分支计算距离，其中垂直速度的偏差需按固定比例系数参与范数计算。若计算出的距离小于给定的共振宽度与热速度的乘积，则判定为共振状态，设置共振标志为真并跳出当前循环。
