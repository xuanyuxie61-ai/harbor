# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

缺失的第一个方法用于计算给定 y 坐标处的三维磁场矢量。该部分需要实现 Harris 电流片模型的磁场分布：x 方向的磁场分量需要根据双曲正切函数进行计算，并受基础磁场幅度和特征尺度参数控制；y 方向磁场分量为零；z 方向磁场分量为常数导向磁场。此外，计算前需要对输入的 y 坐标进行数值溢出安全检查，如果绝对值过大则抛出异常，最后将三个分量组合返回。

缺失的第二个方法用于计算平衡态磁场 x 分量关于 y 坐标的二阶导数。此处需要实现基于 Harris 平衡方程的二阶导数推导，算法上涉及计算双曲正切函数与双曲正割平方函数的乘积，并需要结合基础磁场幅度以及特征尺度的平方进行缩放。

缺失的第三部分代码用于估算磁重联率和能量释放。首先，设定经典 Spitzer 电阻率为 1e-6，入流速度为 1e3。接着，基于 Sweet-Parker 模型估算重联率，该计算涉及电阻率、特征尺度和出流速度（取为中心点的阿尔芬速度），需要在出流速度大于零时进行计算并输出，否则输出提示；随后，基于 Petschek 模型计算快速重联率的典型理论参考值，公式中包含圆周率与常数 1e2 的自然对数运算。在能量估算部分，需计算电流片的磁能密度（基于基础磁场幅度的平方与真空磁导率计算），再通过最大 y 坐标与特征尺度估算体积和总储能，最后将总能量除以 1e25 以换算等效的 X 级太阳耀斑级别，并将上述各项物理量逐行打印输出。
