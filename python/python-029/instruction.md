## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个量子散射计算程序，主要实现分波法求解散射问题。缺失的代码分为三个部分：

第一部分：计算散射矩阵与相移
功能：在匹配半径处，利用内部波函数与外部自由粒子的波函数进行对数导数匹配，求出散射矩阵元和相移，并对波函数进行归一化。
算法思路：首先计算匹配半径处的外部波数与半径的乘积，调用已实现的球贝塞尔函数及其导数接口，获取Riccati-Bessel函数的值及其导数，进而组合出出射和入射的Hankel函数及其导数。然后，利用内部波函数在匹配点处的对数导数，与外部Hankel函数进行对数导数匹配，此处需要实现对数导数匹配方程以求解散射矩阵元。接着，根据散射矩阵元计算相移，并将其归约至特定的主值区间内。最后，对计算得到的内部波函数进行归一化处理，归一化因子由波函数绝对值的最大值决定。

第二部分：计算散射振幅
功能：根据已求得的散射矩阵，计算特定散射角度下的散射振幅。
算法思路：采用分波展开法计算散射振幅。将输入的角度转化为其余弦值，并初始化散射振幅数组。计算散射振幅的公共前因子，该前因子与波数成反比且包含虚数单位。随后，遍历所有可能的轨道角动量和总角动量。对于自旋为二分之一的粒子，总角动量的取值需根据轨道角动量是否为零来决定取一个或两个特定值。在循环中，从字典中提取对应的散射矩阵元，计算该总角动量的简并度，调用已实现的勒让德多项式接口，按分波展开公式累加计算散射振幅，最后乘以前因子并返回。

第三部分：分波循环求解与数据收集
功能：遍历各个分波，调用径向薛定谔方程求解器，收集各分波的物理量并打印结果。
算法思路：对轨道角动量和总角动量进行双重循环，总角动量的取值规则同第二部分。在每次循环中，调用已实现的径向薛定谔方程求解函数，并传入相关物理参数及网格参数。将求解返回的散射矩阵元、相移和波函数以轨道角动量和总角动量构成的元组为键存入相应的字典中，同时保存径向网格。最后，按照特定格式打印当前分波的轨道角动量、总角动量、相移、散射矩阵元的绝对值以及吸收率。
