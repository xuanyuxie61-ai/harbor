## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个关于核裂变或原子核形变等物理过程的模拟，主要涉及裂变碎片的质量分配、分布宽度计算以及最终质量分布的统计与可视化。项目已有的其他函数默认已正确实现，不需要重新定义。

缺失的代码主要分为三个部分，具体功能与算法思路如下：

第一部分：轻、重碎片质量数的计算。该部分需要根据形变变量和集体坐标中的不对称参数，利用精确的双曲线关系来分配轻碎片和重碎片的质量数。采用此关系是为了避免在大形变情况下质量数越界。计算出初步结果后，需要对其进行边界保护，确保轻、重碎片的质量数被限制在1到总质量数减1的范围内，最终以浮点数形式返回。方向提示：此处需要实现基于双曲线关系的质量分配方程，并加入边界截断处理。

第二部分：碎片质量分布宽度的计算。该部分需要根据势能曲率（即势能二阶导数在轻、重碎片对应极小值处的绝对值）和系统温度，计算碎片的分布宽度。计算时需对曲率设置一个极小的下限以防止除零错误。随后，需要将此宽度从集体坐标维度转换为质量维度上的宽度，转换时需用到总质量数以及从collective_coordinates模块导入的不对称参数。方向提示：此处需要实现由势能曲率和温度决定的统计涨落宽度计算模型，并完成集体坐标宽度到质量宽度的标度转换。

第三部分：质量分布直方图的构建与归一化。该部分需要对模拟得到的所有最终质量数据进行统计。首先确定质量统计的范围，下限取1和总质量数20%中的较大值，上限取总质量数的80%。然后在该范围内基于给定的箱数构建直方图。最后，根据总模拟轨迹数和每个区间的宽度对计数进行归一化，以得到每单位质量的概率密度。方向提示：此处需要实现直方图统计与密度归一化逻辑。
