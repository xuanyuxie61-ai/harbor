该项目涉及暗物质直接探测中的核反冲物理模拟。需要补全的代码包含三个核心函数：

第一个函数用于计算Helm形状因子的平方。算法思路为：首先处理反冲能量小于等于0和质量数非正的异常边界情况。接着，根据质量数计算三个核参数（核半径参数、表面厚度参数、皮肤深度），单位均为飞米，其中核半径参数由质量数的立方根按经验公式计算并减去0.60，表面厚度参数设为0.52，皮肤深度参数设为0.9。然后，计算等效核半径（需结合上述三个参数），并从飞米转换为米。将反冲能量转为焦耳后，根据动量转移与核质量和反冲能量的关系计算动量转移，并处理动量转移接近0的除零边界。最后，计算动量转移与等效核半径、皮肤深度的乘积，调用一阶球贝塞尔函数，根据Helm形状因子模型方程进行计算，并返回结果的平方。

第二个函数用于计算暗物质核反冲的微分事例率。算法思路为：首先根据质量数近似计算核子质量，并求出暗物质与核子的约化质量。接着，调用已实现的形状因子函数、最小速度函数以及速度积分函数。将截面单位由皮靶转换为平方厘米。然后，根据Lewin & Smith标准暗物质探测文献中的前置因子方程计算核心项（需结合局部暗物质密度、截面、质量数平方、形状因子、暗物质质量与约化质量平方）。为了将单位转换为文献常用的 events/(keV·kg·day)，需乘以一个约为1.0e-15的综合单位转换常数。最后，对非有限或负数的结果置零，并返回浮点数结果。

第三个函数用于基于拒绝采样法生成暗物质反冲事例。算法思路为：在达到目标事例数或最大尝试次数前进行循环。在给定的能量范围内均匀采样试探能量，调用已实现的微分率函数计算该能量处的率。如果当前率超过初始设定的最大率，则将最大率更新为当前率的1.2倍并跳过本次尝试。生成一个在0到最大率之间的均匀随机数，若该随机数大于当前计算得到的率，则拒绝本次采样并继续循环。
