## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个关于相对论重离子碰撞的物理模拟程序，主要包含初始条件产生和流体力学演化两个部分。

首先，程序需要计算碰撞的参与者密度分布。此处需要实现光学Glauber模型中的参与者密度计算逻辑。该部分会用到两个原子核的厚度函数、核子-核子碰撞截面以及两个原子核的质量数，需要按照特定的指数幂次形式来计算并返回总的参与者密度。

接着，程序需要根据算出的参与者密度来生成初始能量密度分布。在此步骤中，需要调用已有的函数获取参与者密度分布，求取其最大值，并将整个分布进行归一化处理，使得能量密度的峰值设定为15 GeV/fm³。随后，利用生成好的初始能量密度分布，调用二维流体力学演化函数，并传入空间坐标网格等参数，获取随时间演化的能量密度、温度和比熵的历史数据。

最后，在流体力学演化的时间循环中，需要实现能量密度的时间演化步骤。此处需要采用显式欧拉方法来求解相对论流体力学中的能量密度演化方程。该演化由阻尼项和扩散项两部分组成，其中阻尼项与声速的平方、当前能量密度和固有时间有关，扩散项与扩散系数及拉普拉斯算子有关。更新能量密度时，需结合上一步的能量密度及时间步长进行迭代。为了保证数值稳定性，需将更新后的能量密度限制在一个极小值和极大值之间。完成演化计算后，将新的能量密度、通过已有函数转换得到的温度以及计算得到的比熵分别存入对应的历史记录数组中。
