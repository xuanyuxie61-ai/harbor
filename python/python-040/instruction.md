本项目涉及粒子物理中的共振态寻找与蒙特卡洛模拟，缺失的代码主要包含以下三个部分的功能：

第一部分：实现计算某种粒子共振态传播子的函数。该函数接收一个代表能量平方的参数，若该参数小于或等于零，则返回实部和虚部均为零的复数。否则，需要根据模型对象提供的质量和总宽度，计算该共振态的布雷特-维格纳（Breit-Wigner）传播子分母。在计算时，为了避免在共振极点附近发生数值溢出，需要对分母的绝对值设置一个极小的下限阈值进行保护性截断处理，最终返回分母的倒数。

第二部分：基于给定的共振态参数，生成符合相应质量分布的蒙特卡洛事件样本，并模拟探测器效应。该函数采用拒绝采样法，首先根据传入的随机数种子初始化，并在指定的质量范围内进行采样。计算采样包络线的峰值高度时，需取共振峰处的概率密度值，并设置一个极小的下限以防数值下溢。在循环中，在质量范围内均匀随机抽取质量值，计算其对应的布雷特-维格纳概率密度，并与另一个均匀分布的随机数比较以决定是否接受。一旦接受，需模拟探测器的质量分辨率展宽效应：将质量分辨率比例设定为1.5%，对质量施加均值为零、标准差为该比例乘以质量的高斯抖动。确保展宽后的质量仍在合法范围内，最终返回生成的事件质量数组。

第三部分：在给定的不变质量谱和对应计数分布中，通过滑动窗口法寻找最优的共振信号位置及其统计显著性。算法遍历不变质量谱中的每个质量点作为候选中心，以该中心定义一个指定半宽度的信号窗口，以及一个位于信号窗口外侧、宽度为两倍信号窗口宽度的侧边带。利用侧边带内的事件数，按窗口与侧边带的面积比外推估计信号窗口内的背景计数。计算信号窗口内的超出事件数（实际计数减去背景估计，且不小于零），并根据超出事件数与背景估计的平方根之比计算统计显著性。最终，记录并返回使显著性最大的中心质量值、对应的超出事件数以及最大显著性值。
