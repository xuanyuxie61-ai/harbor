# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一维波动方程的正演模拟与基于伴随法的全波形反演梯度计算，同时包含了频率域的完美匹配层（PML）边界处理。缺失的代码主要分布在三个部分：

1. 第一部分缺失的是正演过程中状态变量时间导数的空间离散与边界处理。在内部节点，需要使用中心差分格式计算空间二阶导数。在边界节点，需要根据设定的边界类型进行分支处理：若为吸收边界，需按照一阶吸收边界条件进行单向波的一阶差分离散；若为反射边界，则需将边界导数直接置零。

2. 第二部分缺失的是伴随方程的导数计算函数内部逻辑，以及反演梯度的计算与平滑过程。在伴随导数函数中，伴随场变量的空间离散与边界处理方式与正演部分一致，并需要将观测数据残差作为源项叠加到伴随场的时间导数中。函数外部，需要调用已有的四阶龙格库塔积分器对伴随方程进行反向时间积分，获取时间序列后将其反转为正向时间。随后，根据波动方程的伴随理论计算对波速的梯度：先对正演波场求取空间二阶导数，再将其与伴随场在时间轴上进行积分相乘，并乘以与波速立方成反比的系数。最后，为了提高数值稳定性，需对计算出的梯度进行空间平滑，平滑规则为中心点权重占一半，左右相邻点各占四分之一，边界点则保持原值。

3. 第三部分缺失的是频率域波动方程带PML吸收边界的系统矩阵构建。首先需要根据PML原理定义复数拉伸因子，该因子由阻尼系数和角频率构成。然后基于中心差分格式填充复数类型的系统矩阵，差分系数的计算需引入复数拉伸因子的平方，矩阵的对角线元素还需额外叠加上波数的平方项。

请根据上述提示，在项目代码中找到合适的缺失位置，补全这些逻辑的实现。已有的其他辅助函数（如时间积分器等）默认已正确实现，无需重新定义。
