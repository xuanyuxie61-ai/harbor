## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

片段一：该部分实现了一个根据流函数推导速度分量的功能。在二维轴对称或扇形流动中，需要通过流函数计算径向和切向速度。算法思路为：首先计算径向和角度方向的网格平均间距，然后使用中心差分法计算流函数对角度和半径的偏导数，在网格边界处需退化为单侧差分处理。最后，根据流函数与速度分量的极坐标关系方程（注意包含半径倒数的因子）计算并返回两个速度分量。

片段二：该部分实现了一个极坐标下流函数的求解过程。采用雅可比迭代法求解极坐标下的泊松方程。首先初始化流函数矩阵并计算网格间距，进入最大次数为500次的迭代循环。在每次迭代中，更新内部网格点的流函数值，计算时需考虑极坐标拉普拉斯算子中径向和角度方向的差分系数以及中心系数。更新后强制施加边界值为0的无滑移边界条件，并判断全场最大误差是否小于1e-8以决定是否提前退出迭代。迭代收敛后，利用已有的物理类实例调用片段一所述的流函数与速度关系方法，推导并返回速度场。

片段三：该部分实现了一个温度场的显式时间推进步骤。基于显式时间步进格式，需要调用已有的物理方法计算温度场的极坐标拉普拉斯项（代表扩散效应）和平流项，然后结合时间步长、热扩散率以及源项，对温度场进行更新。更新后需强制施加边界条件：内侧边界的CMB温度，外侧边界的表面温度，以及角度方向的周期性边界条件。最后，需将全场温度截断在表面温度和CMB温度的物理范围内，以确保数值符合物理约束。
