# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个地磁场发电机数值模拟与诊断系统。缺失的代码主要包含四个部分：

1. **均值场发电机方程右端项计算**：此部分负责计算环向场和极向场的演化率。需要遍历所有的球谐模式，针对每个模式，分别在非均匀径向网格上计算环向场和极向场的扩散项（包含基于通量的二阶空间导数计算）与衰减项。对于环向场，需加入由α效应产生的源项；对于极向场，需加入由差速自转剪切产生的Ω效应源项。两部分均需施加径向边界值为零的条件，最后按偏移量将各模式结果组装进全局右端项向量中。涉及径向网格步长及半径平方的中间变量需仔细处理。

2. **自适应时间积分循环**：此部分实现了一个嵌入式高阶龙格-库塔时间推进算法。在循环中，需计算七个阶段的斜率，并利用不同阶次的权重系数分别计算高阶与低阶的解。通过两者的差值评估误差，若误差大于阈值，则按比例缩小时间步长并重试；若误差可接受，则更新时间与状态变量，记录数据。当误差极小时，需按比例放大时间步长以提升计算效率。

3. **地磁诊断量计算**：此部分根据球谐系数提取地磁场的偶极子特征。需从系数字典中获取一阶零次和一次的实部与虚部高斯系数，进而计算偶极子倾角、磁偏角、归一化偶极矩以及相对自转轴的倾斜角。计算反三角函数时，需设置极小值阈值以防止除零错误，并保证反余弦函数的输入值被限制在有效范围内。

4. **代理模型反转频率估算**：此部分实现一个轻量级代理模型，根据输入的磁扩散率和α效应参数估算磁场反转频率。需先计算无量纲的发电机数，其中包含差速自转特征值、特征长度等预设常数；随后基于超临界度（即发电机数减去临界常数，如10）按经验线性比例（如0.05）估算反转频率，并确保结果非负。
