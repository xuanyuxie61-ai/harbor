你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

项目主要涉及地球物理断层位移正演计算与反演求解，缺失的代码包含以下四个核心部分：

1. 第一个缺失部分是一个类的私有方法，用于计算走滑断层分量对三维位移矢量的贡献。该方法基于Okada (1992)模型的简化公式。计算时需要先求取空间点到源点的距离参数，为了避免除零错误，需设定一个极小值作为该距离的下界。接着，利用该距离参数及空间坐标，分别计算三个方向的位移近似值，计算过程中涉及反三角函数与有理分式的组合，最后将三个位移分量组装为数组返回。

2. 第二个缺失部分是一个类方法，用于计算网格中每个三角形单元的形心坐标。方法需要先初始化一个与单元数量对应且维度为2的全零数组用于存储结果。然后遍历所有单元，根据当前单元的节点索引提取对应节点的坐标，计算这些节点坐标在列方向上的均值，将其赋给结果数组，最后返回该数组。

3. 第三个缺失部分是一个独立函数，用于实现Tikhonov正则化最小二乘求解。该函数需要根据输入的设计矩阵、权重矩阵、观测数据、正则化参数和正则化算子，构建正规方程的左端矩阵和右端向量。构建左端矩阵时，需注意正则化参数需取平方后再与算子矩阵的转置乘积相结合。为保证数值稳定性，需调用项目中已有的确保矩阵正定的函数对左端矩阵进行处理。之后，求解线性方程组得到模型参数，并计算近似后验协方差矩阵（若矩阵求逆失败则使用伪逆），最后返回模型参数和协方差矩阵。

4. 第四个缺失部分是一个类方法，用于实现带有L1范数正则化的非线性反演，采用Nelder-Mead优化算法。该方法需要处理初始模型参数的默认值，并根据权重矩阵计算其平方根形式。核心是定义目标函数，该目标函数由三部分构成：加权残差的二范数平方项、Tikhonov正则化二范数平方项以及L1范数正则化项（模型参数绝对值之和乘以给定权重）。随后，实例化项目中已有的优化器类并执行优化，最后使用项目已有的有限性检查函数验证最优模型参数，并返回最优模型、最优目标函数值及函数评估次数。
