该部分代码主要涉及三维重力正演建模与反演网格构建，包含三个关键功能模块：

第一部分：计算均匀密度长方体在观测点处产生的重力异常。算法思路上，需要通过遍历长方体的八个顶点组合，利用符号权重进行解析解的累加计算。此处需要实现均匀长方体重力异常的正演解析方程，方程中包含对数项和反正切项。在实现时，必须注意数值稳定性，需要对计算出的距离及特定对数项的真数设置极小值下限以避免奇点，并对反正切项的参数进行范围截断以避免数值溢出。最终累加得到的结果需要乘以万有引力常数、密度值以及一个用于将单位转换为毫伽的系数（其值为10的5次方）。

第三部分：构建三维反演的离散化网格。算法思路上，需要通过嵌套循环生成规则三维网格的中心坐标列表和对应的体积数组。参数赋值如下：x、y、z三个方向的网格数量分别为8、8、6；x和y方向的网格间距均设置为150公里，z方向的网格间距设置为500公里。在坐标生成时，x和y方向的坐标需以网格中心为零点向两侧展开，z方向的坐标向下为负值表示地下深度。每个网格单元的体积即为三个方向间距的乘积。
