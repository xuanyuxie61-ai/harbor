## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个基于谱方法的准地转流体动力学求解器。缺失的代码主要包含三个部分：

第一部分：计算位涡方程右端项的函数。该函数需要实现准地转位涡方程的右端项计算，算法思路为组合非线性平流项、行星涡度梯度项、双调和粘性耗散项以及线性阻力项。非线性平流项需通过调用已实现的雅可比算子计算；行星涡度梯度项需要流函数在x方向的偏导；耗散项和阻力项分别需要流函数的双调和算子和拉普拉斯算子。计算过程需在谱空间求导再转换至物理空间。参数方面，需使用行星涡度梯度参数、粘性系数和阻力系数。此外，函数需接收一个可选的随机强迫参数，若该参数不为空，则将其叠加到右端项结果中。相关的物理场与谱场转换、雅可比计算等辅助函数已默认实现。

第二部分：求解器状态的初始化代码片段。该片段需要将给定的总流函数赋值给求解器对象，然后通过快速傅里叶变换将流函数转换至谱空间，并利用求解器自带的亥姆霍兹算子将谱空间的流函数转换为物理空间的位涡场，赋值给求解器对象。最后，将求解器用于存储历史右端项的列表初始化为长度为3且元素全为空的列表。

第三部分：生成物理空间随机强迫场的函数。该函数需要实现基于能量后向散射理论的随机强迫生成。算法思路为：在谱空间中，强迫的振幅由特定的能量谱函数决定，该函数与波数成负三分之五幂律并受截断函数限制，计算时需引入能量输入率参数和时间步长，并对零波数进行除零保护。超过截断波数一定倍数的波数其能量谱需置零。生成强迫场时，需在谱空间生成复高斯白噪声，实部和虚部均由独立的标准正态分布生成并组合。为了保证逆傅里叶变换后结果为实数，需对复噪声强制执行Hermitian对称性，即对零频分量及偶数维度对应的最高频率分量强制取实部。最终，将谱空间的强迫振幅与复噪声相乘，逆变换回物理空间并取实部返回。相关的截断函数和正态随机数对生成函数已默认实现。
