## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及海洋内波动力学模型的计算与图结构构建。缺失的代码主要由三个部分组成：

第一部分：实现内波色散关系和群速度的计算功能。首先，针对输入的水平波数和垂直波数，计算两者的平方和作为分母，并设定极小值阈值以避免除零。此处需要实现内波色散关系方程，根据浮力频率和科氏参数计算角频率的平方，确保其非负后开方得到角频率。其次，在群速度计算中，需调用已实现的色散关系函数获取角频率，分母变为波数平方和的平方，同样需处理极小值阈值。此处需要实现内波群速度方程，计算水平和垂直方向的群速度分量，注意它们在计算因子、波数与角频率比值及正负符号上的差异。

第二部分：实现特定条件下的内波频率近似计算。此处需要计算固定的水平波数（由两倍的圆周率与1000.0的比值得到）以及基于模态和水深参数的垂直波数（由模态数、圆周率与水深计算得到）。随后，此处需要实现内波色散关系方程的近似形式，计算得出对应的角频率。

第三部分：构建图结构的边权重以模拟内波的空间传播代价。对于网格中的节点，计算其向右、向下及对角方向的邻居节点索引。在水平和垂直传播中，需根据局部水深计算垂直波数（圆周率与水深绝对值最大值，且水深绝对值下界设为1.0），结合固定水平波数，此处需要实现内波群速度分量的近似计算方程。计算得到的群速度分量需保证绝对值具有下界（水平分量下界为0.01，垂直分量下界为0.001），边权重由空间步长与群速度分量的比值确定。对角线方向的传播权重则由对角线空间步长与水平和垂直群速度分量最大值的比值计算得出。
