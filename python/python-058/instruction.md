# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

1. 缺失了一个用于计算反应-扩散方程右端项的方法。该方法需要接收两个表示状态变量的数组，并返回它们随时间的变化率。算法思路是：首先分别计算这两个变量的拉普拉斯项（采用9点差分格式，假设已有相关方法可直接调用）；然后计算反应项，对于第一个变量，其反应项是一个非线性项，涉及该变量自身、第二个变量以及参数alpha、beta、epsilon的运算，其中epsilon作为倒数系数；对于第二个变量，其反应项为两个变量的差值。最后，将各自的扩散项（第一个变量的扩散系数为Du，第二个变量的扩散系数为Dv并附加一个delta系数）与反应项结合，得到最终的时间导数。

2. 缺失了对流动力学模型的初始化、时间积分及能量计算代码。需要实例化一个对流动力学类，传入网格尺寸和间距变量，并指定以下物理参数：alpha赋值为0.25，beta赋值为0.001，delta赋值为1e-5，epsilon赋值为0.002，Du赋值为5.0e3，Dv赋值为1.0e2。随后，将时间步长设为0.5以满足数值稳定性约束，积分步数设为40。调用该类的积分方法获取最终的两个状态变量，并调用对应方法计算总对流能量。

3. 缺失了一个计算二维水汽通量辐合（仅水平分量）的函数。此处需要实现二维水汽通量辐合公式。算法思路是：首先校验输入的水汽混合比、两个方向风速、空气密度等数组的形状是否完全一致；然后计算x和y方向的水汽通量（密度、水汽与对应风速的乘积）；接着使用中心差分方法分别计算x方向通量的x偏导数和y方向通量的y偏导数；最后将两者相加并取负值得到辐合结果，并对结果中的非有限值（如NaN或Inf）替换为0.0。

注意：项目中已有的其他辅助函数（如9点差分拉普拉斯计算、中心差分梯度计算等）均已正确实现，无需重新定义。
