# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失片段实现了一阶Rosenbrock方法（线性隐式欧拉）的单步求解，用于推进浓度随时间的变化。算法思路为：计算当前浓度的右端项和对角雅可比矩阵近似，构建基于对角近似的线性隐式系统来求解浓度增量，最后用当前浓度加上增量得到新浓度。数值处理上，需要对线性系统的分母进行极小值阈值判断以防止除零，同时对新浓度设置极小值截断以保证浓度非负的物理意义。此处需要实现形如基于对角雅可比近似的隐式欧拉方程求解。

第二个缺失片段负责对单个网格单元执行化学步。算法思路为：根据输入的网格单元索引，提取该单元的温度、特定物种（空气分子）浓度以及光解速率，利用这些参数实例化平流层化学求解器对象并配置光解速率，随后调用前述的Rosenbrock单步求解方法，得到化学反应后的新浓度。

第三个缺失片段用于计算全局网格每个单元内的输运源汇项，包含扩散和平流过程。算法思路为：遍历所有网格单元，提取几何坐标、高度、风速及扩散系数。垂直扩散项采用中心差分近似，需根据网格经纬度数量推算垂直相邻层的索引，并处理顶层与底层的边界越界情况；经向扩散项在此处简化为零；垂直平流项采用一阶迎风格式，根据垂直风速的正负选择向后或向前差分。最终将各项累加为总源汇。注意实现中需包含高度从千米到米的单位转换以及垂直网格步长的计算。
