## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

该项目是一个基于球面网格的能量平衡模型（EBM）的动力学模拟系统。

片段一：计算球面能量平衡模型的温度倾向（即微分方程的右端项）。
功能与算法：首先将输入温度转换为双精度浮点数，并基于网格顶点坐标计算各节点的纬度。随后，依次调用已有函数获取太阳辐射、冰反照率、吸收的短波辐射和射出长波辐射。接着，计算球面拉普拉斯算子以表征热扩散过程，并基于纬度计算热容量。加入火山与太阳外强迫后，需要进行物理边界处理：对温度进行截断（下限为180，上限为350），基于截断后的温度重新计算长波辐射；同时，若截断温度低于200，则强制右端项非负；若高于340，则强制右端项非正。最终返回短波、长波、扩散项与强迫的代数和除以热容量的结果。

片段二：构建模型在平衡温度处的雅可比矩阵（线性化矩阵），用于系统稳定性分析。
功能与算法：需要计算局地反馈导数。其中，反照率对温度的导数需通过实现逻辑斯蒂函数的导数来获取（提示：参考冰反照率的温度跃迁，平衡温度为263.15，反照率高值与低值分别为0.65和0.08）；长波辐射对温度的导数需实现斯特番-玻尔兹曼定律的导数。将短波与长波反馈导数的差值除以热容量常数（值为2.5e8）填入矩阵的对角线。接着处理扩散耦合项，耦合系数由扩散系数除以（2.5e8乘以1e6）得到。假设节点按一维环状拓扑连接，需将耦合系数加到相邻的非对角线元素上，并从相应的对角线元素中减去，同时处理首尾节点的闭环连接。

片段三：计算球面网格上的全球能量净辐射不平衡量的积分。
功能与算法：基于网格顶点计算纬度及其正弦值，进而计算第二类勒让德多项式P2。利用P2计算太阳辐射的空间分布（提示：此处需实现包含P2项的纬度分布函数，对应系数为0.482）。调用已有函数计算冰反照率和射出长波辐射，得到短波吸收与长波发射的差值作为辐射不平衡密度。最后，调用已有的网格积分函数将该不平衡密度在球面网格上进行面积分并返回。

注意：项目中已有的其他函数（如计算反照率反馈、长波辐射、拉普拉斯算子、热容量、网格积分等）均默认已正确实现，无需重新定义，直接调用即可。
