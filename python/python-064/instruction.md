这是一个关于气候能量平衡模型的数值计算项目，主要涉及反照率反馈计算、时间步进积分以及稳态求解。项目中有三处代码缺失，具体功能与算法思路如下：

第一处缺失为一个计算冰-海洋反照率的函数。该函数需要根据输入的温度值，利用Sigmoid函数在冰面反照率和海洋反照率之间进行平滑过渡计算。过渡宽度设定为5.0K，并以冰点温度作为中心阈值。计算完成后，需要将反照率结果裁剪限制在海洋和冰的反照率极值范围内，同时需保证函数能够根据输入是标量还是数组返回对应类型的数据。

第二处缺失位于时间积分的主循环中，需要实现单步的物理过程与隐式求解构建。首先，需要根据当前温度分段计算反照率：当温度低于263.15K时反照率取0.62，高于273.15K时取0.25，在这两个温度阈值之间则进行线性插值。随后，计算吸收的短波辐射，并结合Budyko线性长波辐射模型计算向外长波辐射，从而得到净辐射强迫。最后，构建隐式时间积分的系统矩阵与右端项，系统矩阵需综合包含热容项除以时间步长、扩散项以及Budyko反馈项，其中反馈项需按热容矩阵对角元素均值进行缩放，右端项同样需对强迫项进行相应的缩放处理。

第三处缺失为一个利用牛顿迭代法求解稳态温度的函数实现。在迭代过程中，需调用已有的反照率和长波辐射计算函数，计算当前温度下的辐射平衡残差，若残差绝对值小于容差则提前返回。若未收敛，则使用有限差分法（扰动步长设为0.01）近似计算残差对温度的偏导数（即雅可比矩阵），为防止除零错误，需对极小导数值设置1e-15的下限。接着根据牛顿法更新温度，并将更新后的温度限制在200.0K至350.0K的安全范围内，当温度更新量小于容差时认为迭代收敛。
