# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个基于有限单元法的对流-扩散-反应方程求解及参数反演框架。缺失的代码主要包含以下三个部分：

1. 局部单元矩阵计算：在某方法中，需要计算一维单元的长度并进行正值校验。接着，计算该单元的局部质量矩阵与初始扩散刚度矩阵。为了抑制对流占优时的数值振荡，需根据流速、扩散系数和单元长度计算Peclet数，并引入人工扩散系数（需用到双曲正切函数与Peclet数的代数关系进行修正）。使用修正后的系数更新刚度矩阵，并计算对流矩阵以及与质量矩阵成比例的反应项矩阵（比例系数包含反应常数和延迟因子）。最后返回质量矩阵，以及刚度、对流与反应矩阵之和。

2. 正演模型计算：定义一个接收模型参数向量的正演函数，参数依次为扩散系数、流速和反应常数。需对参数进行物理范围校验，若不满足非负或正数要求，则返回全为极大惩罚值的数组。在异常捕获结构中，调用项目中已有的瞬态求解器进行求解，取结果最后若干时间步的均值作为稳态近似，提取观测点位置的值并返回；若求解过程发生异常，同样返回极大惩罚值数组。

3. Levenberg-Marquardt优化迭代：实现LM算法的迭代求解过程。在最大迭代次数的循环中，计算当前参数的残差和均方根误差并记录，调用已有的有限差分方法计算雅可比矩阵和梯度，若梯度范数小于设定容忍度则终止。构造LM法方程的系数矩阵（近似海森矩阵加上阻尼系数乘以对角线元素构成的正则项），求解得到参数更新增量。对更新后的参数进行合法范围裁剪，并计算新参数的均方根误差。若误差减小，则接受新参数并缩小阻尼系数；否则保留原参数并放大阻尼系数。
