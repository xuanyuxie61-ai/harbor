# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个基于大涡模拟（LES）的三维不可压缩流体求解器。缺失的代码分为三个部分，分别对应亚格子模型、流场时间推进算法以及两者的耦合调用。项目中已有的辅助函数（如计算应变率张量、空间导数、拉普拉斯算子等）默认已正确实现，可直接调用。

1. 第一部分缺失代码需要实现Smagorinsky亚格子尺度（SGS）模型。该部分需要根据输入的应变率张量分量，计算滤波宽度（其值等于网格三个方向间距乘积的立方根），并计算应变率张量模。为了防止数值计算发散，需要对张量模进行截断处理，将其上限限制为1e6。随后，利用滤波宽度、张量模和Smagorinsky常数计算亚格子粘度，并最终根据亚格子粘度与应变率张量的关系推导出亚格子应力张量的六个独立分量。该函数需返回亚格子粘度以及包含这六个应力分量的元组。

2. 第二部分缺失代码需要实现三维不可压缩Navier-Stokes方程的分数步（投影）法时间推进。首先在预测步中，计算流场的对流项、扩散项和体积力，从而得到中间速度场。接着，需要求解压力Poisson方程以获取压力修正量，此处需要采用基于相邻六点空间离散的迭代方法，迭代次数设定为50次。最后在校正步中，利用求得的压力修正量对中间速度场和原压力场进行更新，得到下一时间步的速度和压力。

3. 第三部分缺失代码为主循环中的耦合逻辑，需要将亚格子模型与流场求解器结合。此处需要调用上述的Smagorinsky模型计算亚格子粘度（注意传入的Smagorinsky常数值应为0.18）。大涡模拟的亚格子效应通过增加有效粘度来体现，因此需要将分子粘度与亚格子粘度的全局平均值相加，得到有效粘度。最后，将该有效粘度作为参数传入分数步求解器中，完成单步时间推进。
