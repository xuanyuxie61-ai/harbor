该项目是一个多物理场耦合的相场模拟系统，主要涉及相场、温度场和浓度场的演化计算。需要补全的代码分为三个部分：

第一部分缺失的是相场方程右端项的计算逻辑。该部分算法思路为：首先对输入的序参量进行物理范围截断以保证数值稳定性；接着，基于截断后的序参量分别计算四部分驱动力：扩散项（调用五点差分格式的拉普拉斯算子并乘以参数的平方）、化学驱动力（调用双势阱函数的导数并取负）、热耦合项（与温度偏差和特定多项式项相关）以及溶质耦合项（与浓度偏差和同样的多项式项相关）；将这四部分加和并除以弛豫时间得到初步右端项。最后，若存在速度场，需加入基于一阶迎风格式计算的对流修正项，即根据速度方向选择后向或前向差分计算序参量梯度，与速度点积后从初步右端项中减去。

第二部分缺失的是温度场和浓度场方程右端项的计算逻辑。对于温度场，算法需分别计算热扩散项（对温度场求拉普拉斯并乘以热扩散系数）、对流项以及潜热源项，并将三者线性叠加，其中潜热源项与当前及上一时刻的序参量有关。对于浓度场，算法需计算变系数扩散项（先根据序参量求出有效扩散系数，再计算带变系数的拉普拉斯）、对流项以及溶质排出源项，将三者线性叠加。

第三部分缺失的是系统时间推进主循环中的单步积分与数值限制逻辑。该部分需依次推进三个物理场：首先推进相场，通过龙格-库塔法进行时间步进，步进后需将序参量截断至物理合理范围；其次推进温度场，计算其右端项后，需限制其变化率绝对值不超过时间步长倒数的10倍以防止数值爆炸，然后进行显式欧拉时间推进，并将结果温度截断至熔点温度上下一定范围内（下限低2.0，上限高1.0）；最后推进浓度场，同样限制其变化率绝对值不超过时间步长倒数的5倍，进行显式欧拉推进后，将浓度截断至0.0到1.0的物理区间内。
