该项目涉及流固耦合（如圆柱涡致振动）的数值模拟。缺失的代码主要分为三个部分：

1. 第一部分用于计算流体作用在结构上的气动力系数（阻力系数与升力系数）。需要根据圆柱表面的涡量分布，利用数值积分（如梯形积分）方法沿圆柱表面积分。积分时需利用三角函数（正弦和余弦）对表面涡量进行加权求和。角度步长由圆周率与表面离散点数计算得出，同时需用到来流速度和圆柱半径作为归一化参数。该部分最终返回阻力系数与升力系数。

2. 第二部分用于将力系数转化为实际物理力并推进结构运动。首先，此处需要根据动压公式计算动压，其中流体密度设为1.0，并需用到来流速度和圆柱直径。随后，将动压与第一步得到的力系数相乘得到物理阻力与升力，并将其组合成二维外力向量。接着，调用结构对象的步进方法传入该外力以更新结构状态，并将当前时间步的位移和速度保存到对应的历史记录数组中。

3. 第三部分用于隐式求解二维结构动力学方程，获取下一时刻的运动状态。需将外力与非线性平均力叠加得到综合作用力。针对两个自由度的运动，需循环构建隐式时间积分的线性方程组。此处需要实现隐式积分格式（如Newmark-beta法等）的方程组构建，生成包含时间步长、质量、刚度和阻尼影响的系数矩阵，以及包含当前位移、当前速度和综合作用力影响的常数向量。通过求解该线性方程组，得到下一时刻的位移和速度，并按顺序存入长度为4的新状态数组中。
