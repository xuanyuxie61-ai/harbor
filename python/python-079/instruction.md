# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

该项目是一个海洋工程领域的浮体水动力学计算程序，主要包含波浪运动学计算、浮体运动方程求解以及波浪外力计算。缺失的代码分为三个部分：

1. 缺失一个用于计算Airy线性波运动学量的函数。该函数接收空间和时间坐标、波幅、周期、水深以及方向角等参数，返回包含波面升高、水平和垂向速度及加速度、动压力的字典。在此函数中，重力加速度应设定为9.80665，需采用深水近似方法计算波数，并计算相应的双曲函数项。为防止分母为零异常，需对双曲函数分母设置1e-12的极小值阈值进行保护。计算动压力时，海水密度需设定为1025.0。此处需要实现Airy线性波理论的相关方程。

2. 缺失浮体时域运动方程的右端项计算逻辑。该部分需计算浮体运动的加速度，并将速度与加速度拼接返回。此处需要实现多自由度浮体运动方程，综合考虑总质量矩阵、辐射阻尼矩阵、静水恢复力矩阵以及外部激振力向量的平衡关系，并通过求解线性方程组获得系统加速度。

3. 缺失一个计算时变波浪力的函数。该函数接收时间和浮体状态向量作为输入，返回六自由度波浪力向量。在此函数中，需要调用前述的波浪运动学计算函数，其中计算深度设定为-10.0，波幅设定为1.5，水深设定为100.0，方向角设定为0.0。基于Froude-Krylov与Morison简化模型，利用波浪水平速度乘以3.0e5的系数得到纵荡力，垂直速度乘以5.0e5的系数得到垂荡力，并将纵荡力乘以5.0的力臂作为纵荡力矩，其余自由度力默认为零。此处需要实现波浪外力的简化计算模型。
