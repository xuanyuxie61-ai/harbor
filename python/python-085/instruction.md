该项目是一个基于有限元方法的平面应变力学与接触分析求解器。缺失的代码包含以下三个核心方法：

第一个方法用于计算平面应变弹性矩阵。该方法需要根据材料的杨氏模量和泊松比这两个参数，实现平面应变条件下的本构方程，构建并返回一个3x3的弹性矩阵。提示：矩阵的构建依赖于一个由杨氏模量和泊松比组合的系数因子，以及由泊松比构成的非对角线项和剪切项。

第二个方法用于计算三角形单元上三个线性形函数的梯度。该方法接收单元索引，通过获取该三角形单元的三个节点坐标，首先计算三角形面积的两倍作为分母（若该值过小需抛出退化单元异常），然后根据等参元的几何映射关系，分别计算三个形函数关于x和y的偏导数，最终返回一个形状为(3, 2)的梯度数组。提示：此处需要实现二维线性三角形形函数梯度的解析表达式。

第三个方法用于组装接触力向量。该方法需要根据当前位移场，结合接触节点处的法向间隙和切向滑移量，计算法向接触力和切向摩擦力，并将其组装到全局力向量中。算法思路为：首先判断法向间隙是否小于设定阈值以确定激活集；对于激活的接触节点，计算法向压力（需保证其非负），并计算试算切向力，随后根据库仑摩擦定律（摩擦系数与法向压力的乘积作为极限）对切向力进行裁剪限制，最后将法向和切向接触力分配到全局接触力向量对应的自由度上。提示：此处需要实现增广拉格朗日接触算法与库仑摩擦模型。
