## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个基于壳单元的非线性有限元求解器，主要包含内部力计算、牛顿-拉夫逊迭代求解以及弧长法控制。

缺失的代码片段1：功能为计算有限元模型的内部节点力向量。算法思路为：遍历所有单元，提取单元节点与自由度映射，获取单元位移；利用已有的膜应变-位移矩阵和弯曲应变-位移矩阵计算应变与曲率；通过材料本构矩阵和弯曲本构矩阵计算膜力与弯矩；此处需根据虚功原理，将单元的内力贡献（基于B矩阵转置、截面内力与单元面积）累加组装到全局内部力向量中。

缺失的代码片段2：功能为使用牛顿-拉夫逊法求解非线性平衡方程。算法思路为：在最大迭代次数内，计算当前位移下的内部力与残差；组装切线刚度矩阵（包含线性刚度与几何刚度）；施加Dirichlet边界条件（底部节点全自由度固定，顶部节点仅限制面内位移允许轴向位移，底部首个节点施加z转动约束）；提取自由度进行降维求解位移增量，若矩阵奇异则添加微小对角正则化项辅助求解；通过线搜索（Armijo条件）确定步长并更新位移；计算系统势能；依据力残差和位移增量的相对范数判断收敛；根据是否收敛返回包含位移、收敛状态、迭代次数、残差范数、能量及历史记录的结果字典。

缺失的代码片段3：功能为实现弧长法（球面准则）的修正步计算。算法思路为：计算当前预测步的残差与切线刚度矩阵；针对自由度降维系统，分别求解关于残差向量和外力向量的两组线性方程组，得到对应的位移增量；此处需实现弧长法的球面约束方程，计算约束残差；利用上述两组位移增量、当前预测步与初始步的位移差及载荷因子差，计算载荷因子修正量；进而组合得到位移修正量，最后返回自由度维度的位移修正与载荷因子修正。
