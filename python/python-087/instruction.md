## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及柔性多体动力学系统的建模与仿真，包含模态分析、受约束系统的积分算法以及广义力的计算。需要补全的三个代码片段功能及思路如下：

片段1的功能是计算柔性梁截断至指定模态数的模态质量矩阵与模态刚度矩阵。算法思路上，需要先通过调用已有的求根方法获取指定模态数下的特征根，并据此计算波数；随后基于欧拉-伯努利梁理论，利用材料密度和截面积构造对角化的模态质量矩阵，利用弹性模量、截面惯性矩以及波数的四次方关系构造对角化的模态刚度矩阵。

片段2的功能是实现受约束动力学系统的时间积分推进，属于一种带约束的辛积分算法（如类似Velocity Verlet的半步更新法）。算法思路为：首先计算当前时刻的拉格朗日乘子与外力；然后对广义动量进行半步更新，更新时需结合约束雅可比矩阵的转置与乘子的乘积；接着通过调用求解质量矩阵相关方法完成位置的全步更新；随后对位置进行一步牛顿法投影以满足约束；最后基于新时刻和新位置重新计算乘子与外力，完成剩余半步的动量更新。

片段3的功能是计算系统的广义力向量，涵盖内力、阻尼力与外力。算法思路为：先获取组装好的刚度矩阵，并采用瑞利阻尼模型构造阻尼矩阵（此处质量比例系数赋值为0.1，刚度比例系数赋值为0.001）；接着计算由刚度和阻尼引起的恢复力与耗散力；随后施加外力，包括对各个刚体施加重力（重力加速度取9.80665，按刚体自由度偏移定位并施加在z方向上），以及对首个柔性构件的第一阶模态施加简谐激励（按刚体自由度总数计算偏移量定位，激励频率设为10.0，幅值设为0.05）。
