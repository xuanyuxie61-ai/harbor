本项目涉及有限元分析中考虑混凝土徐变、老化及收缩效应的计算模块。缺失的代码分布在三个不同的函数中，具体功能与算法思路如下：

第二处缺失：实现平面应力状态下的本构矩阵构建。首先，需要计算一个缩放因子，其值等于弹性模量除以（1减去泊松比平方）。随后，基于该因子构建一个3x3的对称矩阵：主对角线前两个元素为1，第三个元素为（1减去泊松比）除以2；副对角线非零元素为泊松比；其余元素为0。最后返回该矩阵。

第三处缺失：实现时步分析中徐变与收缩效应的有限元等效计算流程。具体步骤如下：
