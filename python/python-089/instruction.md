本项目涉及二维平面应力问题的有限元动力学分析与随机振动响应计算。缺失的代码主要由三个功能模块组成：

第一个缺失片段负责二维线性三角形单元的刚度矩阵与一致质量矩阵的计算及全局组装。算法思路为：遍历所有单元，提取节点坐标并计算面积（需跳过面积小于极小阈值1e-14的退化单元）；接着计算形函数导数构造应变-位移矩阵，以及平面应力本构矩阵（若杨氏模量为向量则取单元节点均值，并依赖泊松比参数）；此处需要实现单元刚度矩阵和一致质量矩阵的方程，最后将单元矩阵根据自由度映射累加至全局矩阵中并返回。

第二个缺失片段负责模态分析与频响函数（FRF）的模态叠加法求解。算法思路为：首先求解广义特征值问题获取特征值与特征向量（优先使用SciPy，若不可用则回退至NumPy结合质量矩阵求逆缩放的方法）；计算自然频率并根据指定的模态数进行截断；此处需要实现模态振型关于质量矩阵的归一化方程；随后，基于模态参与系数（由激励与响应自由度对应的振型乘积决定）和阻尼比，此处需要实现频响函数的模态叠加计算方程并返回。

第三个缺失片段负责随机振动分析。算法思路为：调用前述频响函数后，此处需要实现输出功率谱密度（PSD）的计算方程；通过对输出PSD进行积分求得均方位移响应及均方根，同时利用二阶谱矩与零阶谱矩的比值计算零点穿越频率（需处理均方位移极小接近0时除零的异常情况）；最后，在一阶共振频率附近定义积分核函数，调用已有的柯西主值积分函数进行验证并打印各项计算结果。
