## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个三维电磁场的FDTD（时域有限差分）仿真程序。缺失的代码共包含三个关键部分：

第一部分：实现三维空间中矢量场旋度的计算功能。算法思路是利用有限差分法，分别计算旋度的三个空间分量。对于网格内部点采用中心差分格式，对于边界点采用单侧差分格式。计算过程中需要用到三个方向的空间步长参数。此处需要实现矢量场旋度的数学计算逻辑。

第二部分：属于仿真引擎类中的一个方法，用于更新电场分量。算法思路为首先提取当前磁场分量并调用函数计算其旋度，随后利用更新方程计算下一时间步的电场。该更新方程考虑了介质的损耗特性（隐式处理），使用了包含介电常数、电导率和时间步长信息的预先计算好的衰减与更新系数。此处需要实现含损耗的安培定律更新方程的离散形式。

第三部分：仿真的主时间步进循环。算法思路按执行顺序包括：1. 调用已有方法执行标准的FDTD磁场和电场更新；2. 分别对电场和磁场应用PML（完美匹配层）吸收边界条件；3. 应用PEC（理想导体）边界条件；4. 推进时间步计数和物理时间，并施加激励源；5. 数值稳定性保护，对六个场分量数组进行软截断，将绝对值超过最大限制（如1e6）的值截断至该阈值，同时检测并清除非有限数值（Inf或NaN），将其置为零；6. 按照设定的采样间隔，计算系统的总能量与功率损耗，并将它们及当前时间记录到历史列表中。
