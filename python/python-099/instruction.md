本项目涉及电磁波在等离子体多层膜中的反射特性计算。缺失的代码分为三个部分：

第一部分：实现Drude模型的复介电常数计算功能。
算法思路：首先对输入的电子密度、碰撞频率和波频率进行数值下限保护，确保电子密度不小于1.0，碰撞频率和波频率均不小于1e-3。接着，调用已有的函数获取等离子体频率，并计算其平方值与包含波频率和碰撞频率的分母项。此后，需要依据Drude模型方程，分别计算介电常数的实部和虚部。最后增加一个数值保护逻辑：如果计算出的实部小于-1e6，则将其强置为-1e6，最终返回由实部和虚部构成的复数。注意：获取等离子体频率的函数已默认存在，无需重新定义。

第二部分：实现基于传输矩阵法（TMM）的多层膜反射系数计算功能。
算法思路：首先将输入的折射率和厚度转换为对应数据类型的数组，并校验二者形状是否一致，若为空则返回0。计算真空波数，并将入射角限制在0到约1.5708（即二分之派减1e-6）之间，设定入射介质折射率为1.0。根据极化方式（TE或TM）计算入射介质的导纳。初始化一个2x2的单位复数矩阵作为全局传输矩阵，随后遍历每一层：应用斯涅尔定律计算该层折射角的正弦值（需将其虚部保留，实部限制在-1到1之间以防越界），进而求出折射角和余弦值；根据极化方式计算该层导纳和相位厚度；构建该层的2x2特征矩阵，并与全局传输矩阵进行矩阵乘法累积。遍历结束后，设定基底为真空并计算其导纳。最后，利用全局传输矩阵的四个元素以及入射和基底的导纳，依据电磁场边界条件计算总复反射系数（除法运算请使用已有的安全除法函数）。

第三部分：实现给定频带下的反射率光谱计算逻辑。
算法思路：在遍历频带的循环中，先将当前频率转换为角频率（值为2乘以派再乘以频率）。初始化各层的复折射率数组，对每一层调用第一部分的Drude介电常数计算函数，获取复介电常数后对其开平方以得到该层的复折射率。随后，调用第二部分的多层膜反射计算函数，设定入射角为0，极化方式为TE，得到总复反射系数。接着调用已有的功率反射比计算函数得到反射率，将复反射系数和功率反射比分别存入预定义的光谱结果数组中对应的索引位置。
