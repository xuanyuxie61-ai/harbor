# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及电磁场的有限元分析计算，包含以下三个缺失的代码部分：

1. 缺失了一个计算磁阻率的类方法（磁阻率定义为磁场强度与磁通密度的比值），其结果用于有限元刚度矩阵的组装。算法思路：调用类中已有的方法获取磁场强度，然后计算其与输入磁通密度的比值。在计算时必须防止除零错误：当磁通密度的绝对值小于极小阈值（如1e-14）时，需将其替换为保留符号（需加上极小偏移以避免零的符号）且大小等于该阈值的值，最后返回安全的比值结果。

2. 缺失了有限元全局刚度矩阵与载荷向量的单元组装循环逻辑。算法思路：遍历网格的所有单元，并根据标签过滤器跳过不需要的单元。对于每个目标单元，获取其节点索引、形函数梯度与面积；计算单元质心坐标，并在质心处调用磁阻率函数与源函数求值。接着，构建一个3x3的单元刚度矩阵，其每个元素等于磁阻率、面积和对应形函数梯度点乘的乘积；构建单元载荷向量，依据P1元的体积积分公式，其值等于源项值、面积与三分之一系数的乘积。最后，将单元矩阵和向量的结果分别累加或追加到全局载荷向量和稀疏矩阵的行列索引及数据列表中。

3. 缺失了一个由磁矢势计算径向磁通密度的方法。算法思路：此处需要实现柱坐标系下径向磁通密度的计算方程（提示：该物理量等于磁矢势与半径的比值，加上磁矢势对半径的导数）。在离散计算中，假设单元内磁矢势为线性插值，导数用相邻节点差分与单元长度的比值表示。对于内部节点，其结果取左右相邻单元在各自质心处计算值的平均；对于两端的边界节点，则使用单侧单元的差分在质心处进行计算。网格的节点坐标和单元长度等属性可通过类成员获取。
