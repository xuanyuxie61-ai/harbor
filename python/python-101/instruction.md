# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失片段是一个计算布拉格光栅解析反射率的函数。该函数需要首先验证光栅长度非负。接着，根据耦合系数绝对值的平方与失谐参数平方的差值（即判别式），算法需要分为三种情况求解反射率方程：判别式大于零对应带隙内解，需使用双曲函数，并注意当双曲函数变量过大时直接返回1以避免数值溢出；判别式近似为零对应带隙边缘解，使用代数形式；判别式小于零对应带隙外解，需使用三角函数。计算得出分子与分母后，需处理分母接近零的边界情况，并确保最终返回的反射率被截断在0到1的闭区间内。

第二个缺失片段是一个利用数值方法（线性打靶法）求解耦合模方程并计算光栅反射率的函数。同样需要验证光栅长度非负。函数内部首先需定义一个微分方程右侧函数，调用项目中已有的耦合模方程函数（设置无损耗）。然后，利用已有的四阶龙格-库塔(RK4)求解器，从两个正交的初始条件（分别为前向入射和后向入射）进行积分。获取积分终点处的后向振幅后，计算线性组合系数以满足光栅终端的边界条件。将两组轨迹按系数组合得到真实的前向与后向模振幅，最后在入射端根据前后向振幅的比值计算数值反射率，同样需截断在0到1之间。该函数需返回空间坐标、前向模、后向模以及数值反射率。

第三个缺失片段是项目主程序中用于耦合模理论与布拉格光栅模拟的验证脚本。首先需调用已有的函数输出主节标题。接着定义物理参数：耦合系数kappa赋值为5000的复数形式，失谐参数定义为从负两倍kappa到正两倍kappa之间的5个等距采样点，光栅长度赋值为100微米。打印这些参数信息后，遍历失谐参数数组，分别调用前述的数值求解函数和解析函数计算反射率，并求两者的绝对误差，以格式化表格的形式打印出归一化失谐、数值反射率、解析反射率和误差。最后，输出一个小节标题用于有限差分时域耦合模传播，调用已有的FDTM函数（失谐设为0），并打印前向模在终点的振幅大小以及后向模在起点的振幅大小。
