# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

1. 需要补全一个定义矩形纳米柱截面相对介电常数分布的方法。该方法接收坐标变量以及纳米柱的中心坐标和尺寸参数。算法思路是利用空间几何判定条件，判断给定坐标点是否处于矩形纳米柱内部：若在内部，则赋予硅的介电常数；若在外部，则赋予空气的介电常数。最终返回的结果需转换为复数类型数组。硅和空气的介电常数属性已默认在类中定义，可直接调用。

2. 需要补全一个计算截面上局域有效折射率分布的方法。该方法接收坐标变量及纳米柱的中心坐标、宽度和高度参数。算法思路同样是基于矩形区域的几何判定，对纳米柱内部的区域赋予硅的折射率，对外部区域赋予空气的折射率。相关折射率属性已默认定义。

3. 在部分主流程逻辑中，缺失了调用有限元方法求解散射场以及计算相位延迟的执行代码。首先需调用已有的散射求解函数，为其传入参数：X和Y方向的网格数均为17，计算域X和Y方向的范围均从负1.5微米到正1.5微米，纳米柱中心位于坐标原点，纳米柱的宽度为0.3微米、高度为0.6微米，以此获取电场Z分量及网格节点与单元数据。随后，设定纳米柱中心横纵坐标均为0，宽度为0.3微米，高度为0.6微米，并设定其高度方向的厚度参数为1.0微米。最后，调用已有的相位积分计算函数，传入上述中心坐标、尺寸及厚度参数，从而计算并获取平均相位延迟和透过率。
