## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

该项目主要涉及光脉冲在光纤中传输的非线性薛定谔方程求解及相关物理过程分析。缺失的代码主要分布在三个核心逻辑中：

1. 第一个片段实现的是光纤中拉曼响应函数的生成逻辑。该部分需要将输入的时间数组转换为浮点型并初始化响应数组。根据因果性条件筛选时间大于等于零的部分，对满足条件的区域实现基于双时间常数的拉曼响应模型计算（此处需要实现包含特定前置系数、指数衰减与正弦函数相乘的典型拉曼响应方程）。完成计算后，需要对响应数组进行数值归一化处理，即利用时间步长与数组求和计算积分近似值并进行缩放，防止数值溢出。最后，将归一化后的结果乘以拉曼贡献比例因子并返回。

2. 第二个片段实现的是信号强度与拉曼响应函数的数值卷积运算，用于计算非线性效应中的延迟拉曼项。首先需要计算脉冲电场强度的绝对值平方。为了提高计算效率，采用基于快速傅里叶变换（FFT）的频域卷积方法。在变换前，需要对拉曼响应数组进行零填充以匹配信号长度，从而保证因果性并防止混叠。将信号与响应函数的频域表示相乘后，执行逆傅里叶变换，提取其实部并乘以时间步长，作为最终的卷积结果返回。

3. 第三个片段实现的是调用分步傅里叶方法（SSFM）求解器进行脉冲传播模拟，并对模拟结果进行物理量分析。调用求解器时需传入初始场、时间和各类物理参数，其中需明确设定拉曼贡献比例因子的值为0.18，四阶色散系数设为0.0，开启放大的自发辐射噪声，且不使用隐式算法。求解完成后，利用项目已有的函数分别计算输入与输出脉冲的时域宽度和频谱宽度，同时通过强度求和结合时间步长计算脉冲能量。最后，需格式化打印这些物理量的变化情况，计算并打印能量损耗百分比，同时结合光纤损耗系数与最大传输距离计算预期的含损耗能量，以验证能量守恒及仿真合理性。
