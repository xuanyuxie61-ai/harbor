## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个关于波前传感与重构的系统，主要涉及泽尼克（Zernike）多项式的计算、模态与区域波前重构以及相关数据处理。

缺失的代码主要包含三个部分的功能：

第一部分：泽尼克多项式基的计算函数。该函数需要接收网格大小和最大Noll指标作为参数。首先进行参数校验，确保网格大小至少为3，最大Noll指标至少为1。接着，在归一化的坐标下生成二维网格，并转换为极坐标，同时生成单位圆掩膜。然后，遍历所有的Noll指标，利用已有的转换函数获取径向和角向阶次，计算对应的泽尼克多项式，并将圆外区域的值置为零，将结果存入三维数组中。最后，将数组展平为二维矩阵并返回，同时返回掩膜和坐标向量。

第二部分：基于波前斜率的模态重构函数。该函数接收子孔径的x和y方向斜率、子孔径信息、泽尼克基的展平矩阵以及像素尺度等参数。首先校验斜率数组长度一致。然后，计算每个泽尼克模式在每个子孔径内的平均梯度。这需要从展平的基矩阵中恢复出网格大小，对每个模式提取二维矩阵，利用中心差分法计算其x和y方向的数值梯度。随后，遍历每个子孔径，计算梯度在子孔径区域内的均值，构建交互矩阵。最后，将斜率向量拼接，利用最小二乘法求解出泽尼克系数并返回。

第三部分：主流程中的调用与执行部分。这部分代码首先调用第一部分提到的函数计算泽尼克基，并传入网格大小和模式数。接着，对湍流相位进行泽尼克分解并打印前5个系数。随后，进行模态重构：调用模态重构函数从斜率恢复系数，再利用恢复的系数重建相位。之后，进行区域重构，调用区域重构函数恢复相位。此外，还进行单形约束搜索，调用相关函数寻找最佳泽尼克系数，并设置最大迭代次数为3，打印搜索的最小误差。最后，将湍流的泽尼克系数写入到输出目录下的文本文件中。
