# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

代码片段1：
该片段实现了对联合谱振幅（JSA）矩阵的施密特分解功能。算法思路为：首先对矩阵进行奇异值分解（SVD），提取奇异值并计算其平方作为特征值。当特征值总和大于一个极小的正数阈值时，需对特征值进行归一化处理。此后，需要实现施密特数的计算方程（通过归一化特征值平方和的倒数求得），以及态纯度的计算方程（通过施密特数的倒数求得）。最终，函数需返回特征值、左奇异向量、转置后的右奇异向量、施密特数以及纯度。

代码片段2：
该片段实现了根据量子态纯度计算并发度的功能。算法包含严格的数值校验：纯度参数必须在0到1的闭区间内，若超出允许的微小误差范围则抛出异常，否则将其裁剪限制在该区间内。算法核心为：此处需要实现纯度到并发度的转换方程，即利用1与纯度的差值，经过倍数及开平方运算求得并发度（需确保根号内的值非负）。计算得出的并发度同样需要裁剪至0到1的区间内，并最终返回。

代码片段3：
该片段属于项目的主计算与输出流程。逻辑思路为：依次调用已有的联合谱振幅计算函数、施密特分解函数（对应代码片段1所在函数）、并发度计算函数（对应代码片段2所在函数）以及冯诺依曼熵计算函数。提取这些计算步骤得到的矩阵维度、施密特数、态纯度、并发度与纠缠熵等关键物理量，并将它们格式化打印输出。
