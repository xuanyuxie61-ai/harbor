该项目涉及光在介质中的扩散理论计算，缺失部分主要围绕不同维度和介质结构下的扩散系数求解。

第一部分缺失代码的功能为：计算均匀介质的扩散系数。算法思路是先根据各向异性因子与散射系数计算出约化散射系数，随后计算分母项，该分母为约化散射系数与吸收系数之和的三倍（此处需要实现辐射传输理论中的扩散系数方程）。如果分母小于等于零，需抛出无效光学属性的数值异常；若正常，则返回分母的倒数。

第二部分缺失代码的功能为：实现一维多层介质中的扩散系数计算函数。算法思路是根据输入的深度变量遍历各层边界，定位当前深度所属的层级。定位到层级后，从层级光学属性字典中提取参数，按照第一部分同样的数学逻辑计算约化散射系数与分母。若分母大于零，返回其倒数；若分母非正，或深度超出所有已知层的边界范围，则返回一个极大的惩罚数值（其值设定为10的6次方）。

第三部分缺失代码的功能为：实现二维空间下的扩散系数计算函数。算法思路是将纵坐标变量映射为深度坐标，然后遍历层边界数组以寻找坐标所属的层级。从层级属性列表中提取该层的参数进行相同的约化散射系数与分母计算。同样地，在分母大于零时返回倒数，否则或在坐标超出范围时返回10的6次方的极大惩罚值。
