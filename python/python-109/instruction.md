# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及光脉冲在光纤中传播的数值模拟，主要基于分步傅里叶方法求解广义非线性薛定谔方程。缺失的代码分布在三个不同的功能逻辑中：

1. 第一部分用于计算频域线性算子，此处需要实现包含损耗与各阶色散效应的频域算子。首先将算子初始化为损耗系数的负二分之一。随后，从二阶开始遍历色散系数数组，若当前阶系数绝对值极小（小于1e-30）则直接跳过。对于有效系数，需要将其乘以虚数单位的对应幂次以及角频率的对应次幂，累加至该算子中并返回。

2. 第二部分用于计算非线性效应中的拉曼响应项，此处需要利用快速傅里叶变换实现时域卷积的快速计算。首先将傅里叶变换点数设为原序列长度的两倍，分别对功率序列和响应函数序列进行傅里叶变换。将两者的频域结果相乘后进行逆傅里叶变换，并取其实部完成卷积。最后，截取卷积结果的前原序列长度个点，乘以拉曼分数（其值设定为0.18）及时间步长，将结果返回。

3. 第三部分用于调用脉冲传播主函数，此处需要执行分步傅里叶法的模拟推进。调用时需传入初始振幅、时间序列、目标距离、损耗系数、非线性系数、色散系数数组及中心频率等基本参数。同时需指定拉曼响应参数：拉曼分数设定为0.18，两个特征时间常数分别设定为12.2e-15和32.0e-15。此外，还需设定初始步长为1e-4、记录步数为50，并开启对称分步算法标志。该函数将返回传播位置、振幅和频谱信息。

请注意，项目中已有的其他函数均默认正确实现，你只需根据上述提示找到缺失位置并补全代码逻辑即可。
