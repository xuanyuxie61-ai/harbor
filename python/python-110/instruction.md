# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

这是一个关于量子点-微腔耦合系统（量子光学）的数值模拟项目，主要涉及系统哈密顿量构建、耗散过程跳跃算符构造、物理观测量提取以及Purcell因子的计算。项目中已有的其他函数和物理常数均默认正确实现，不需要重新定义。

缺失代码的功能与算法思路如下：

1. 第一部分缺失代码用于构建系统的哈密顿量矩阵。需要根据给定的光子数截断值确定矩阵维度，系统的基矢由原子的基态或激发态与光子数态直积构成。在矩阵填充时，需要分别计算腔场能量对应的对角元、量子点能量对应的对角元，以及原子与腔场相互耦合的非对角元。此处需要实现Jaynes-Cummings模型的哈密顿量构建逻辑。

2. 第二部分缺失代码用于计算原子激发态的布居数。需要对输入的密度矩阵进行维度校验，确保其为合法的二维数组且维度足够。通过提取密度矩阵中对应原子激发态的特定对角元，并取其实部，即可返回激发态的布居数。

3. 第三部分缺失代码用于构建Lindblad耗散过程中的跳跃算符。首先需要构造原子的自发辐射降阶算符，该算符的作用是使原子从激发态跃迁至基态，同时保持光子数不变；其次需要构造腔场的湮灭算符，该算符使光子数减一，其矩阵元与光子数的平方根有关。此处需要根据给定的截断维数正确构建这两个算符的矩阵表示。

4. 第四部分缺失代码用于计算微腔的Purcell因子。需要对输入的品质因数、有效模式体积和波长等参数进行正数校验。接着计算介质中的波长，最后根据Purcell因子的定义公式完成计算并返回结果。
