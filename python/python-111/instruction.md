# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

缺失部分一：切比雪夫插值求值函数的内部核心逻辑。该函数需要将查询点从原始区间线性映射到[-1, 1]区间，为避免外推导致的不稳定性，需要将映射后的值裁剪限制在[-1, 1]范围内。随后，需采用 Clenshaw 递推算法计算切比雪夫级数的值：初始化一个全零的二维数组（行数为系数数量加2，列数为查询点数量，后两行默认为0），然后从最高阶系数向低阶倒序迭代，按照递推关系更新数组各行的值。最后，通过数组前几行元素和首个系数的特定线性组合计算并返回最终的插值结果。

缺失部分二：一维有限元方法中局部单元刚度矩阵的组装计算。此处需要实现漂移-扩散算子的数值积分，包含扩散项和对流项两部分。首先计算雅可比行列式（其值取为单元长度的一半）以及基于扩散系数和雅可比的乘积因子。在扩散项中，利用形函数导数、该乘积因子和积分权重更新2x2局部刚度矩阵的四个元素。在对流项中，需要计算包含扩散系数、热力学温度、自由能梯度和雅可比的漂移因子，然后利用形函数值、形函数导数、该漂移因子和积分权重，进一步累加更新局部刚度矩阵的四个元素。

缺失部分三：主流程中切比雪夫拟合插值与稳态 Smoluchowski 方程求解的步骤。首先，使用指定阶数（16阶）对自由能剖面进行拟合，获取区间端点和系数；接着，在反应坐标的最小值和最大值之间生成200个精细网格点，分别调用已有的插值函数和求导函数计算对应的自由能及其导数，并将结果组合保存到文件中。随后进入第五步，在相同的坐标范围内构造41个节点的有限元网格，将自由能插值到这些网格节点上，并调用已有的求解函数计算稳态 Smoluchowski 方程的概率密度，其中需指定扩散系数、热力学温度以及左右边界的概率值（分别为0.1和1.0），最后将求解得到的稳态概率密度保存到文件中。
