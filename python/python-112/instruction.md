# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一部分缺失代码实现了Clenshaw-Curtis求积规则的权重计算功能。算法思路是遍历给定的阶数，基于特定角度和三角函数的级数展开来计算每个节点的初步权重。在级数累加过程中，需要根据当前项索引是否达到阶数减一的一半来决定系数取值为1还是2。完成初步计算后，需要根据节点位置（首尾端点或内部节点）对权重进行不同比例的归一化缩放处理，最后返回节点坐标与权重。

第二部分缺失代码用于通过热力学积分方法计算体系的自由能变化。首先，根据给定的温度和特定的玻尔兹曼常数（以kcal/(mol*K)为单位）计算逆温度。接着，调用前述方法获取Lambda积分的节点与权重，并将节点区间从[-1,1]线性映射至[0,1]，权重也做相应缩放。随后，遍历Lambda节点，在构象空间中定义两个积分函数：一个是被积函数，此处需要实现一个特定深度的高斯势阱相互作用模型，计算相互作用能导数并乘以相应的Boltzmann权重；另一个是配分函数积分，仅需计算Boltzmann权重本身。利用已有的稀疏网格积分函数分别计算分子与分母，若分母极小则期望值置零，否则求出各节点处的偏导数期望值。最后，将导数期望值数组与Lambda权重进行内积运算，得到并返回总的自由能变化、Lambda节点及导数数组。

第三部分缺失代码是一个数值积分验证模块。其功能是定义一个三维高斯函数，利用稀疏网格积分在特定维度和层级下计算其在三维超立方体上的积分值。同时，计算该高斯函数的解析积分值（此处需要实现涉及圆周率和误差函数的解析解公式），并打印输出数值积分结果、解析值以及它们之间的相对误差。
