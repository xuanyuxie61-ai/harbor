## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个三维泊松-能斯特-普朗克（PNP）模型的数值求解。缺失的代码主要包含三个部分：

第一部分：实现三维空间中计算物理量（如电势）在内部网格点上的拉普拉斯算子。算法思路是采用有限差分法，在去除边界的内部网格点上，分别沿三个空间方向计算二阶中心差分，并除以对应方向步长（dx, dy, dz）的平方。请勿处理边界点。

第二部分：实现修正泊松方程的迭代求解过程。算法思路为：在最大迭代次数限制内，首先计算移动电荷密度与总电荷密度；接着计算电势的拉普拉斯项以及介电常数梯度引起的修正项，两者结合计算残差；然后使用带有松弛因子的Jacobi迭代法更新电势，为保证数值稳定性，需将更新量裁剪限制在-0.01到0.01之间；之后施加边界条件（z方向首尾层保持原值，x和y方向采用零梯度边界条件）；最后计算相邻两步电势的最大绝对误差，若小于给定收敛阈值则提前退出循环。此处需要实现修正的泊松方程的迭代求解。

第三部分：实现PNP方程的一个完整时间步推进。算法思路是采用Strang算子分裂法以达到二阶时间精度。对于每种离子（如钾离子和钠离子），将一个时间步长的演化分为三步：先执行半个时间步长的扩散过程，再执行一个完整时间步长的电迁移过程，最后再执行半个时间步长的扩散过程。离子的扩散系数和迁移率分别从字典D和mu中获取。此处需要实现PNP方程的Strang分裂时间步进。
