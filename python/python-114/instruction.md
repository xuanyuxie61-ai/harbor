## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

缺失的代码包含四个部分，分别涉及偏微分方程数值解、统计物理能量计算以及模块运行验证：

第一部分缺失一个用于组装一维非线性 Poisson-Boltzmann 方程 Jacobian 矩阵与残差向量的函数。该部分需要采用有限差分法对内部网格点进行离散，通过计算拉普拉斯项、非线性双曲正弦项和电荷源项来构建残差，并依据 Newton 迭代法的偏导数关系构建 Jacobian 矩阵的对角线及次对角线元素。边界需采用 Dirichlet 条件，将边界电势设为零。函数需接收网格点数、当前电势猜测值、电荷密度分布、网格间距、介电常数及 Debye-Hückel 参数作为输入。

第二部分缺失一段基于概率分布计算自由能的代码。此处需要实现基于玻尔兹曼关系的自由能公式，即自由能与概率的对数负相关。为了避免对数计算溢出，需在概率值上加上一个极小的防零常数。计算得到的自由能初始单位为焦耳，随后需要减去最小值进行平移，并除以电子伏特常量将其转换为 eV 单位。涉及的物理常数包括玻尔兹曼常数和电子伏特转换常量。

第三部分缺失一个运行稀疏求解器模块的函数。该函数首先需调用已实现的非线性 PB 方程求解器，获取求解结果并打印收敛状态、Newton 迭代次数、最终残差范数及电势极值。随后，需构建一个测试用的三对角矩阵，使用已有的带状 LU 求解器类进行带状转换、矩阵分解和求解，以验证求解器的正确性，并打印计算残差。

第四部分缺失一个运行能量景观模块的函数。该函数首先需生成模拟的蛋白质二面角数据，包含两个主要构象态及高斯噪声。然后调用已实现的自由能表面构建函数，并打印降维坐标范围、局部极小值个数及能垒高度。接着，在网格上使用已实现的分片线性插值函数对自由能进行插值验证，并计算有效点比例。最后，调用已实现的单形顶点坐标生成函数构造指定维度的正则单形，验证其顶点间距离是否符合理论距离公式。
