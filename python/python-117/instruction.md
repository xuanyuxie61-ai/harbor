## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目包含三个核心功能模块，分别涉及物理场方程的离散化求解、分子间相互作用力计算以及数值寻根算法。

第一个缺失片段的功能是构建一个离散线性系统，用于求解包含边界条件的物理场。算法思路为：此处需要实现球对称泊松-玻尔兹曼方程的转化，通过引入中间变量，将其转化为对称的一维亥姆霍兹方程。随后在径向网格上进行差分离散化，构建严格对称的三对角系数矩阵与右端项。在处理边界条件时，为了维持矩阵的对称性，需要将边界点的耦合贡献转移到右端项中。数值参数方面：网格步长平方与参数平方的乘积用于构建内部节点的对角线元素；表面势由表面电荷、元电荷、介电常数和纳米粒子半径计算得出；边界点的变量值由半径与表面势的乘积得到；内部节点矩阵对角线及次对角线需赋予特定的正负浮点数值以翻转符号保证正定；左右端点需单独设置对角线数值，并清除边界耦合项的数值，右端项需累加已知边界值的贡献。

第二个缺失片段的功能是计算粒子间的范德华相互作用力。算法思路为：此处需要实现Lennard-Jones势函数对距离的导数。为了防止距离趋近于零时的数值奇异性，当距离小于特定阈值时，需截断常规计算，替换为基于距离差值的谐波排斥力计算。数值参数方面：截断距离阈值设定为0.1；谐波排斥的刚度系数设定为1000.0；在常规力计算中，需计算特征尺寸与距离比值的6次方和12次方项，并乘以一个数值为24.0的前置系数。

第三个缺失片段的功能是基于Chebyshev多项式插值的代理根查找算法，用于寻找目标函数在给定区间内的实根。算法思路为：首先在区间内计算Chebyshev极值点并进行函数值采样；接着调用已有的系数计算函数获取Chebyshev系数，并根据尾部范数与最大系数绝对值的相对关系进行高阶项截断；随后利用截断后的系数调用已有的伴随矩阵构建函数，通过求解该矩阵的特征值来定位根；最后，根据虚部相对容差和区间外容差筛选出实根，并将符合条件的根映射回原始区间，对结果进行去重与升序排序。数值参数方面：插值阶数、系数截断相对阈值、复根虚部相对容差和区间外根容差均作为输入参数；截断判断需从高阶向低阶遍历；极值点和根的区间映射均涉及区间中点与半长的线性组合计算；去重精度需保留至小数点后12位。已有其他函数默认已正确实现，不需要重新定义。
