# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

代码片段1：
该片段实现分子动力学模拟中嵌入原子方法（EAM）的嵌入力计算功能。算法思路为：首先计算原子i和j的嵌入能对密度的导数；接着计算原子j和i的密度对距离的导数，这需要用到基于截断半径的密度函数及其导数，并考虑截断半径处的平滑处理；最后根据链式法则将两者结合得到嵌入力，并更新对应原子的受力和系统的维里应力。此处需要实现EAM势函数中基于密度梯度的嵌入力方程。涉及的参数包括：截断半径、不同原子类型的密度函数参数及密度梯度参数、嵌入能导数等。

代码片段2：
该片段实现Nose-Hoover链（NHC）恒温器的速度缩放因子计算功能，采用多时间步积分方案。算法思路为：首先根据系统当前动能与目标温度的差值计算热浴力；然后在内循环中，从最外层向最内层反向更新热浴速度的半步值，内层更新需考虑与外层热浴速度的耦合关系；利用最内层热浴速度半步值计算并累加缩放因子；最后正向更新各层热浴速度。此处需要实现NHC恒温器的运动方程积分。涉及的参数包括：主时间步长、内循环步数、热浴链长度、各层热浴质量参数、目标温度及玻尔兹曼常数等。

代码片段3：
该片段实现从界面高度涨落谱拟合界面能（或界面刚度）的功能。算法思路为：首先过滤掉非正的波矢和谱值，对波矢和谱取对数后进行线性最小二乘拟合；根据毛细波理论，对数谱与对数波矢呈线性关系，斜率应为负二，截距包含热力学能与界面能比值的对数信息；最后通过截距反推并返回拟合的界面能。此处需要实现毛细波理论对应的对数线性拟合方程。涉及的参数包括：波矢数组、涨落谱数组、玻尔兹曼常数和系统温度。

代码片段4：
该片段实现计算界面粗糙度的简化响应面模型功能。算法思路为：从输入参数数组中提取温度、浓度和界面能，计算粗糙度的近似值，该值正比于热力学能与界面能比值的平方根，并乘以一个与浓度相关的线性修正函数。此处需要实现界面粗糙度随温度与界面能标度关系的物理模型。涉及的参数包括：温度、浓度、界面能，以及一个取值约为8.617333e-5的玻尔兹曼常数，同时需对界面能添加微小量以防止除零。
