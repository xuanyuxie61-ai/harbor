# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失片段是一个用于计算给定位置势能值的方法。该方法首先需检查模型系数是否已拟合，并将输入的坐标点转换为浮点数组，确保其形状为三维坐标。计算时，需将坐标点相对于参考点进行中心化和尺度缩放处理，然后根据预存的幂次列表与对应系数，实现势能面的多项式泰勒展开方程，通过累加各阶项得到所有点的势能数组并返回。

第二个缺失片段是一个计算所有可能事件（吸附、脱附、扩散）速率的方法。该方法需实现阿伦尼乌斯形式的速率方程。参数定义上，每个表面位点对应3种事件；需设定尝试频率为10的13次方每秒；热力学能的计算需将温度乘以玻尔兹曼常数再除以基本电荷常数，以得到电子伏特单位的值。对于每个位点，需获取其位置并评估局域能量。如果位点为空，则计算吸附事件速率，其活化能取0与负位点能量的较大值；如果位点被占据，则计算脱附事件速率，其活化能取0与位点能量加1.0的较大值，同时计算扩散事件速率，其扩散势垒固定为0.3电子伏特。

第三个缺失片段是一段执行随机反应动力学（Gillespie算法）的调用与测试逻辑。需要实例化随机反应动力学类，传入表面对象、势能面对象以及500.0开尔文的温度。接着初始化一个全零的整型物种映射数组，其长度与表面位点总数一致，并将索引为0和1的位点物种分别设为1和2。随后调用Gillespie步进方法执行一步模拟，获取时间间隔、事件类型和位点索引，格式化打印这些结果，最后返回积分器对象。
