## 一、科学问题概述

本项目构建了一个**从光感受器层到神经节细胞层的端到端多尺度计算模型**，模拟视觉刺激在视网膜中的处理过程。核心科学问题是：

> **如何建立从光子捕获到脉冲发放的完整视网膜神经编码计算框架，并定量分析各层级（光感受器→双极细胞→神经节细胞）的信息转换效率？**

视网膜是中枢神经系统中研究最为透彻的神经网络之一，其计算复杂性体现在：
- **多层非线性处理**：光转导级联、突触传递、侧向抑制
- **时空编码**：空间感受野结构与时间脉冲模式的耦合
- **大规模耦合网络**：数十亿神经元通过化学突触和电突触互联

本模型通过融合15个种子项目的核心算法，构建了一个可运行的博士级科学计算平台。

## 三、新增数学物理模型与核心公式

### 3.1 光转导级联ODE系统

光感受器（视锥/视杆细胞）将光信号转换为电信号的分子机制：

```
d[PDE*]/dt = α·I_light - β·[PDE*]

d[cGMP]/dt = α_gc_max / (1 + ([Ca²⁺]/K_gc)^{n_gc}) - γ·[PDE*]·[cGMP]

d[Ca²⁺]/dt = -η·I_Ca
```

其中光电流：

```
I_Ca = g_max · [cGMP]³ / ([cGMP]³ + K_cGMP³) · (V_m - E_Ca)
```

参数说明：
- `PDE*`：激活的磷酸二酯酶浓度（光子激活的酶）
- `cGMP`：环磷酸鸟苷浓度（控制离子通道）
- `Ca²⁺`：胞内钙离子浓度（负反馈调节）
- `α, β`：PDE激活/失活速率常数
- `K_gc, n_gc`：鸟苷酸环化酶的钙半激活浓度和Hill系数

### 3.2 光适应稳态方程（有限差分）

光感受器外段中的光适应可用稳态扩散-反应方程描述：

```
D·∇²C(x,y) + S(x,y) = 0
```

中心差分离散（Jacobi迭代）：

```
C_{i,j}^{(new)} = (C_{i-1,j} + C_{i+1,j} + C_{i,j-1} + C_{i,j+1} + h²·S_{i,j}/D) / 4
```

### 3.3 Gray-Scott突触反应-扩散方程

突触间隙中神经递质的时空演化：

```
∂U/∂t = D_u·∇²U - U·V² + F·(1-U)
∂V/∂t = D_v·∇²V + U·V² - (F+K)·V
```

其中：
- `U`：神经递质前体/囊泡密度
- `V`：突触间隙中游离神经递质浓度
- `D_u, D_v`：扩散系数
- `F`：补充速率（feed rate）
- `K`：清除/重摄取速率（kill rate）

**9点Laplacian stencil**（O(h⁴)精度）：

```
L = (1/(6dx²)) · [ 1   4   1 ]
                  [ 4 -20   4 ]
                  [ 1   4   1 ]  * A
```

### 3.4 差异高斯（DoG）感受野模型

双极细胞的中心-周边拮抗结构：

```
RF(x,y) = A_c·exp(-(x²+y²)/(2σ_c²)) - A_s·exp(-(x²+y²)/(2σ_s²))
```

### 3.5 Legendre正交基展开

感受野调谐曲线的一维Legendre展开：

```
f(x) = Σ_{k=0}^{N} c_k · P_k(x),  x ∈ [-1, 1]
```

系数计算（Gauss-Legendre求积）：

```
c_k = (2k+1)/2 · ∫_{-1}^{1} f(x)·P_k(x) dx
    ≈ (2k+1)/2 · Σ_i w_i · f(x_i)·P_k(x_i)
```

Legendre三项递推：

```
P_0(x) = 1
P_1(x) = x
P_k(x) = [(2k-1)·x·P_{k-1}(x) - (k-1)·P_{k-2}(x)] / k
```

### 3.6 三角插值基函数

周期神经发放模式的三角基：

```
τ_k(x) = sin(kπx/2) / [k·sin(πx/2)]        (k为奇数)
τ_k(x) = sin(kπx/2) / [k·tan(πx/2)]        (k为偶数)
τ_k(0) = 1                                  (极限值)
```

### 3.7 非齐次泊松脉冲发放（Thinning）

神经节细胞发放率模型：

```
λ(t) = r_0 + g·max(0, R_bipolar(t))
```

Thinning算法生成脉冲序列：
1. 以速率 `λ_max` 生成齐次泊松候选脉冲
2. 对每个候选脉冲，以概率 `λ(t)/λ_max` 接受保留

### 3.8 CBB网络矩阵与Schur补求解

视网膜神经网络矩阵结构：

```
[ A1  A2 ] [ x1 ]   [ b1 ]
[ A3  A4 ] [ x2 ] = [ b2 ]
```

Schur补：

```
S = A4 - A3 · A1^{-1} · A2
```

求解步骤：
1. 对A1进行带状LU分解
2. 逐列求解 `A1·Y = A2`
3. 计算Schur补S
4. 对S进行稠密LU分解
5. 解 `S·x2 = b2 - A3·A1^{-1}·b1`
6. 解 `A1·x1 = b1 - A2·x2`

### 3.9 Faure准随机序列

基于素数 `p` 的p进制展开生成高维低差异序列：

```
key = Σ_{j=0}^{m} y_j · p^j
quasi(1) = Σ_{j=0}^{m} y_j / p^{j+1}
```

高维通过Pascal三角形（模p）变换：

```
y_j^{(new)} = Σ_{i=j}^{m} y_i^{(old)} · C(i,j)  (mod p)
```

### 3.10 球谐函数（视网膜曲面展开）

```
Y_l^m(θ, φ) = N_l^m · P_l^{|m|}(cosθ) · exp(i·m·φ)

N_l^m = sqrt((2l+1)/(4π) · (l-|m|)! / (l+|m|)!)
```

### 3.11 Lanczos log-Gamma近似

```
ln Γ(z) ≈ 0.5·ln(2π) + ln(A_g(z)) + (z-0.5)·ln(z+g-0.5) - (z+g-0.5)
```

### 3.12 误差函数近似（Abramowitz-Stegun）

```
erf(x) ≈ 1 - (a1·t + a2·t² + a3·t³ + a4·t⁴ + a5·t⁵)·exp(-x²)
其中 t = 1 / (1 + p·x),  p = 0.3275911
```

## 五、如何运行

### 环境要求
- Python 3.8+
- NumPy

### 运行方式
```bash
cd 125_synth_project
python main.py
```

程序将自动执行以下8个计算模块：
1. **视网膜几何离散化**：生成127个六边形感光细胞，执行Delaunay三角化，评估网格质量
2. **光感受器光转导**：求解光适应稳态方程，模拟光转导ODE级联
3. **双极细胞感受野**：构建DoG感受野，执行Legendre基展开
4. **突触反应-扩散**：模拟Gray-Scott神经递质时空演化
5. **神经节细胞编码**：分析三角插值基、生成泊松脉冲、计算信息效率
6. **大规模网络求解**：构建CBB矩阵，使用Schur补方法求解
7. **视觉刺激生成**：生成光栅/blob/漂移刺激，执行Faure参数采样
8. **数学工具验证**：验证积分、特殊函数的正确性

### 运行时间
- 约 60-80 秒（取决于硬件）
- 主要耗时在Gray-Scott PDE模拟（5000步）和Jacobi迭代（约3000步）

## 六、边界处理与数值鲁棒性

本项目在多处实现了边界保护和数值稳定性措施：

1. **光转导ODE**：状态变量强制非负，cGMP设置最小值保护（1e-10）
2. **Gray-Scott PDE**：浓度场截断到[0,1]，自动调整时间步长满足CFL条件
3. **Legendre多项式**：评估点自动截断到[-1,1]
4. **三角基函数**：x=0处奇点使用极限值τ(0)=1
5. **LU分解**：奇异主元用1e-14替代，防止除零
6. **泊松发放**：发放概率截断到[0,1]，实现不应期机制
7. **矩阵求解**：残差验证确保数值精度

## 七、科学意义与前沿性

本项目模拟了视网膜信息处理的**全链路计算流程**：

1. **空间采样层**：六边形感光细胞阵列 → 几何离散化
2. **光转导层**：光子 → cGMP → 钙离子 → 膜电位（分子级联）
3. **感受野处理层**：中心-周边拮抗 → 空间频率调谐
4. **突触传递层**：神经递质扩散-反应 → 突触后电流
5. **脉冲编码层**：模拟信号 → 泊松脉冲序列 → 信息效率
6. **网络耦合层**：大规模神经元互连 → 线性系统求解

该模型的前沿性体现在：
- **多尺度耦合**：从分子扩散（微米级）到神经网络（毫米级）的统一框架
- **正交基展开**：Legendre/三角基用于神经编码的数学表征
- **信息论分析**：定量评估脉冲编码的信息传输效率
- **高维采样**：Faure序列用于神经参数空间的系统探索
- **矩阵结构利用**：CBB格式求解反映视网膜局部-长程连接拓扑
