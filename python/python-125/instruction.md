第一部分：光转导常微分方程（ODE）模型计算
这部分代码用于计算视觉光转导过程中三个核心状态变量（被激活的磷酸二酯酶PDE*、环鸟苷酸cGMP、钙离子Ca）随时间变化的导数。

第二部分：四阶龙格-库塔（RK4）求解器步进逻辑
这部分代码用于实现RK4算法对上述ODE进行时间步进求解。

- 最后，将当前时间步的推进结果和更新后的状态分别记录到时间数组与状态数组的下一个位置中。
