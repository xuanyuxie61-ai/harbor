## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及偏微分方程的数值求解及生物趋化性模型计算，缺失的代码包含三个部分：

第一部分：实现一个类方法，用于在指定维度上使用隐式向后欧拉法求解一维扩散方程。算法思路是构建对应的三对角系统并调用求解函数。首先根据扩散系数、时间步长和空间步长的平方计算扩散比率。三对角系统的下对角线和上对角线元素均赋为该比率的负数，主对角线元素赋为1加上两倍该比率。同时需应用零诺伊曼边界条件，将首尾主对角线元素修改为1加上该比率。随后沿指定维度对数组的每个切片提取右端项，调用已有的三对角系统求解函数进行求解，将结果存回并返回更新后的数组。

第二部分：实现一个独立函数，用于求解三对角线性系统。算法采用LU分解（追赶法），包含前代和回代过程。输入为下对角线、主对角线、上对角线和右端项四条一维数组，需先检查长度一致性，拷贝并转为浮点型，将下对角线首元素置零。进行LU分解时需注意检查主元是否小于极小阈值以避免除零错误。前代过程更新右端项，回代过程从末尾开始计算解向量并返回。

第三部分：实现一个类方法，用于计算趋化性迁移速度，此处需要实现饱和模型。输入为趋化因子浓度梯度，并包含最大趋化速度和饱和参数两个具有默认值的数值参数。算法需将输入重塑为三维向量，计算其范数，若范数小于极小阈值则返回三维零向量；否则根据饱和模型计算速度因子，该因子由最大趋化速度除以1加上饱和参数乘以梯度范数得到，最后将该因子与梯度向量相乘并返回。
