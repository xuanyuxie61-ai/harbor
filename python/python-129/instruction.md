## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个关于血液凝固级联反应的常微分方程（ODE）系统建模与求解项目，包含生化反应动力学计算、ODE求解及数值稳定性分析。

1. 缺失的第一部分是ODE系统右端项（导数）的计算逻辑。此处需要实现基于米氏方程的酶催化项计算（包含相应的速率常数和米氏常数等参数，注意防止除零的极小值设定），以及基于希尔动力学的纤维蛋白聚合计算。同时需包含凝血酶的正反馈激活逻辑、各种抑制反应（如抗凝血酶和蛋白C系统）及纤溶系统的反应速率。最后，需将所有反应速率按物质守恒定律组装成各物种浓度随时间变化的导数数组。注意各项前特定的系数与衰减常数设定。

2. 缺失的第二部分是Jacobian矩阵刚性特征的分析方法。该方法需接收Jacobian矩阵，计算其特征值，并提取左半平面（实部小于零）的特征值。基于最大绝对值和最小绝对值的负实部特征值，计算系统的刚性比。同时，需根据显式Euler法和梯形法的稳定性边界条件，分别计算这两种数值方法的最大允许步长。注意处理不存在左半平面特征值的边界情况，以及步长公式中的特定常数和防止除零的极小值。

3. 缺失的第三部分是血凝ODE系统求解的执行模块。该函数需实例化血凝网络模型，初始化状态向量（设定特定物种如TF_VIIa、Va、IIa的初始浓度数值，以及将TFPI和tPA设为其总浓度参数值）。随后，调用梯形法求解器，在特定的时间区间和步数下进行积分求解，并打印输出凝血酶的峰值及对应时间、最终纤维蛋白浓度和活化血小板比例。

4. 缺失的第四部分是ODE求解器稳定性分析的执行模块。该函数需实例化稳定性分析器，在一个给定的包含12个维度的典型中间状态点下计算网络的Jacobian矩阵，并调用刚性分析方法输出特征值范围、刚性比及步长建议。此外，还需测试梯形法稳定性函数在一个特定复数点（实部为负，虚部为正）的值，并判断其绝对值是否满足稳定性条件。
