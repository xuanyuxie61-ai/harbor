# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目包含一个多相流反应器仿真系统，主要涉及两相流水力学求解和群体平衡模型。缺失的代码分为三个部分：

1. 第一部分是一个计算两相流局部代数方程残差的函数。该函数接收状态向量（包含气含率和液相速度）和参数字典。你需要提取参数字典中的入口气速，以及入口总体积通量（若未提供，默认其值为入口气速的0.25倍）。同时，需要对状态变量中的气含率进行物理边界约束，将其限制在极小值（如1e-6）至0.95之间。接着，设定漂移通量模型中的分布参数为1.2，特征终端速度为0.23。根据气相速度、气含率和液相速度计算总体积通量。随后，此处需要实现两个残差方程：第一个是基于漂移通量模型的气相速度方程，第二个是总体积通量守恒方程。最后将这两个残差组合成浮点类型数组返回。

2. 第二部分是在反应器类中沿轴向网格逐点求解局部代数方程的逻辑。在遍历轴向网格的循环中，首先根据网格索引计算当前轴向位置，并构建包含液相密度、气相密度、液相粘度、重力加速度、入口气速、轴向位置和参考压力的参数字典。提取当前网格的气含率和液速作为初始猜测值。此处需要调用牛顿求解器来求解局部代数方程，需向其传入残差计算函数、雅可比矩阵计算函数、初始值、收敛容差（1e-9）、最大迭代次数（20）以及状态变量的上下界（气含率下限极小值、液速下限-1.0；气含率上限0.95、液速上限1.0）。求解完成后，将返回的气含率结果再次进行截断保护并更新至类属性，同时更新液速属性。最后，根据更新后的气含率计算气液混合密度，并利用静水压力关系显式计算并更新当前网格的压力值。

3. 第三部分是群体平衡模型中基于求积法计算前四阶矩源项的函数。首先将输入的求积节点和权重转换为浮点数组，并进行异常保护：若节点或权重存在非有限值，或节点值超出合理范围（过大或过小），则直接返回长度为4的全零数组。接着，调用已有的破裂频率函数和聚并核函数，分别计算破裂频率向量和聚并核矩阵。针对0到3阶的每一阶矩，分别计算破裂和聚并的源项：对于破裂源项，死亡项由权重、破裂频率和节点对应矩的乘积求和得到；产生项此处需采用二元均匀破裂近似，当阶数小于等于1时，产生项为2的(1-k)次方乘以死亡项，高阶矩产生项为0。对于聚并源项，需通过双重循环遍历节点对，计算聚并后新生成气泡体积的矩贡献作为产生项，原气泡体积的矩贡献作为死亡项。最终，将破裂和聚并的净源项（产生项减去死亡项）相加，作为该阶矩的总源项存入数组并返回。
