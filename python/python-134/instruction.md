本项目是一个针对质子交换膜燃料电池（PEMFC）膜内水传输的数值模拟程序。项目中需要补全三个与水扩散、残差计算及主流程调用相关的代码片段。

片段1：
功能：计算膜内水扩散系数。
算法思路：基于Springer经验模型，扩散系数由一个与温度相关的基础扩散率和一个与含水量相关的分段函数相乘得到。在计算分段函数时，需要先将含水量限制在0到22的范围内，然后根据含水量所在的不同区间给出不同的线性插值表达式，最后将分段函数的结果限制在0.1到3.0之间，并与基础扩散率相乘返回。

片段2：
功能：计算膜水传输方程的无量纲化残差。
算法思路：针对一维水含量剖面，若为一维数组则计算其二阶空间导数（中心差分法），空间步长由膜厚度与网格点数决定；若非一维则残差置零。在计算源项时，需要在内部网格点集中赋值，该源项模拟电渗拖拽与蒸发效应，其大小呈现为关于网格位置的抛物线分布（中间大两端小）。最终残差由扩散项与源项构成，并使用向量的二范数与水含量剖面的范数之比进行无量纲化处理。注意此处的扩散系数被设定为一个极小的常数。

片段3：
功能：主程序中调用膜内水传输求解器及波动方程验证。
算法思路：首先调用求解膜内水传输的函数（需考虑扩散、电渗拖拽及刚性ODE），传入参数字典，返回稳态水含量剖面及时间网格。随后打印水含量的最小最大值范围、时间步数及总模拟时间。接着进行波动方程解析解验证，在膜厚度范围内生成测试位置点，调用精确波动解析解函数获取各阶导数，并计算波动方程残差（即验证时间二阶导数与波速平方乘以空间二阶导数的差值的最大绝对值），最后打印该残差。
