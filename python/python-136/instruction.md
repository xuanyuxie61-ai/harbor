# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目旨在模拟球形催化剂颗粒内的耦合传质与传热过程，涉及浓度场和温度场的数值求解以及反应动力学的计算。缺失的代码主要包含三个核心部分：

第一部分：球坐标系下浓度场的内部节点与中心边界离散。在内部节点处，需要基于有限体积法实现扩散-反应方程在球坐标守恒形式下的空间离散，计算界面通量与近似体积元，从而组装三对角矩阵的对角线、上对角线、下对角线及右端项，右端项需调用反应速率函数。在球心边界处，利用对称性边界条件进行近似处理，注意此时对角线与上对角线系数的构造均涉及数值6与步长平方的比值。

第二部分：浓度场与温度场的交替求解流程。首先固定温度场求解浓度场，组装对应的三对角矩阵系数，内部节点的离散思路与第一部分类似，但反应项调用特定的动力学模型接口，且浓度和温度在计算前需进行下限截断限制以防数值异常；球心边界同样采用系数6的对称边界处理方式；外表面采用第一类边界条件，对应矩阵系数置为1且右端项赋为表面浓度。求解后需对浓度进行上下限截断并强制赋值表面边界。随后固定浓度场求解温度场，结构类似，但扩散系数替换为有效导热系数，源项变为反应速率与反应热乘积的负数，同样需处理球心对称边界与外表面的定温边界，求解后对温度进行上下限截断并强制赋值表面温度。注意三对角方程求解函数与相关模型接口已在项目中实现，可直接调用。

第三部分：多组分催化反应动力学的速率计算。需要根据温度计算基于阿伦尼乌斯方程的反应速率常数，以及基于吸附热和温度计算两组分的吸附平衡常数。为防止数值溢出，需对吸附平衡常数进行截断限制。随后，基于Langmuir-Hinshelwood型动力学机制计算反应速率，分子为速率常数乘以各组分的吸附项与浓度项的幂次乘积，分母为包含所有吸附项与浓度线性加和的平方。计算时需判断分母是否过小以避免除零错误，并保证最终返回的反应速率满足物理上的非负性约束。
