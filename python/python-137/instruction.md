# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

缺失部分1：计算经典成核率的函数实现。该函数接收过饱和度和温度作为输入。首先需要将输入转换为浮点数组，并进行严格的边界处理：确保温度大于0（若小于等于0则设为极小值1e-6），过饱和度非负；同时需要计算过饱和比及其对数，并处理对数极小的情况以避免除零。接着，计算表面张力，采用基于参考温度的线性修正公式，并确保其值大于0。然后，此处需要实现经典成核理论方程，计算临界成核势垒（吉布斯自由能）及最终的成核率（包含指前因子和指数项）。最后，当过饱和度极小（小于1e-6）时，成核率应强制置为0。

缺失部分2：基于矩方法的种群平衡模型常微分方程组右端项的计算方法。该方法需要从状态向量中提取各阶矩和浓度，根据时间获取当前温度，并调用已有的函数计算过饱和度（确保非负）。利用三阶矩、晶体密度和体积形状因子计算悬浮密度。随后，调用已有的总成核率函数计算成核率，并基于平均尺寸调用已有的尺寸依赖生长速率函数计算生长速率。在构建导数数组时，此处需要实现矩演化方程：零阶矩的导数仅由成核率决定；对于高阶矩，需结合生长项和成核项，并在生长项中引入尺寸依赖的修正因子进行近似计算。最后，此处需要实现质量守恒方程，计算浓度的变化率，该变化率由二阶矩和有效生长速率决定。

缺失部分3：成核与生长模型的对比与验证脚本。首先生成一个从0.01到1.0的过饱和度数组，设定固定温度为320.0 K，悬浮密度为50.0 kg/m³。接着，分别调用已有的初级成核和二级成核函数计算总成核率，并在过饱和度为0.5处打印初级、二级及总成核率的值。然后，调用临界核半径函数并打印结果。对于生长速率的比较，设定测试尺寸为50微米，分别调用并计算幂律模型、尺寸依赖模型、两步模型和BCF螺旋生长模型的速率，传入对应的物理参数（如活化能、指前因子、指数等）并打印。最后，调用生长速率分散函数，设定变异系数为0.15和采样数为1000，计算并打印生长速率分散后的均值和标准差。
