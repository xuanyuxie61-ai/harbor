你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

片段一：该部分用于实现Heston模型特征函数的中间变量计算。算法思路为：首先通过复数运算求解特征方程的根，并对该根的极小值进行截断以避免除零；接着计算一个比例因子，并对该因子的极大值进行截断以防止数值溢出；随后判断是否出现分母极小的退化情况，若退化则采用简化的线性时间表达式计算两个对数特征函数参数，否则使用包含指数项和对数项的完整表达式进行计算，最后将结果转为复数返回。涉及参数包括相关性、波动率系数、均值回归速度、傅里叶变量、到期时间、无风险利率及长期方差等。

片段二：该部分实现了一个基于有限差分法的偏微分方程（PDE）隐式时间步进求解器（用于Heston模型期权定价）。算法思路为：采用隐式欧拉格式构建线性方程组，在每次时间步的倒向迭代中，先计算右端项及边界修正，然后对系数矩阵的边界行（资产价格下界、上界以及方差上界）进行清零，并强制设为对角占优或满足特定差分条件，以施加边界约束。线性系统的求解根据维度规模选择直接求解法或GMRES迭代法（若迭代不收敛则回退至直接法），最后将一维解向量重塑为二维曲面返回。

片段三：该部分为主程序的执行与验证流程。算法思路为：调用Heston PDE定价函数（设定资产价格、方差、时间方向的网格节点数分别为60、30、60），并记录计算耗时；接着，以长期方差的平方根作为波动率输入，计算Black-Scholes模型的参考价格并与PDE价格对比输出差异；最后，调用有限差分法计算PDE下的希腊字母（包括Delta、Vega、Theta、Rho）并输出结果及计算耗时。
