你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及金融数学中远期利率曲线的演化与多因子波动率耦合模型。缺失的代码主要由三个部分组成：

第一部分缺失代码位于均匀网格下的偏微分方程（PDE）右侧项计算函数中。需要实现内部节点的时间导数计算，此处需要实现包含对流项、扩散项、Musiela漂移项和外部强制项的远期利率PDE的离散化。空间一阶和二阶导数需采用中心差分格式。随后，需要基于这些差分离散项构造对应的稀疏有限差分矩阵（建议使用COO格式构建后转换为CSR格式）以及右端强制向量，其中矩阵的每一行需累加各离散项的系数，最后返回时间导数数组、差分矩阵和强制向量。

第二部分缺失代码位于PDE的时间步推进逻辑中。此处需要实现隐式欧拉时间推进格式来求解PDE。需调用右侧项计算函数获取差分矩阵和右端强制向量，构造左端线性系统和右端向量。在求解线性系统前，需修改矩阵和右端向量以施加边界条件：将矩阵的短期利率（首行）和长期利率（末行）对应行清零，主对角线设为1，并将右端向量的短期利率和长期利率分别设定为特定下界值（短期为0.01，长期为0.02，且需与原值取最大值）。最后，使用稀疏求解器求解新状态，并对结果进行截断处理以保证利率非负。

第三部分缺失代码位于多因子波动率的耦合函数中。需要将多个动力系统（如Lorenz、Duffing、Oregonator）的状态归一化并提取为三个特征输入（如取Lorenz状态的二范数归一化，以及其他系统状态的首个分量）。若未提供耦合矩阵，需初始化一个默认的3x3矩阵，其主对角线元素占优（分别约为0.7、0.8、0.8），非对角线元素较小（如0.1或0.2）。需校验传入或默认的耦合矩阵形状是否为因子数乘3。最后，通过矩阵乘法将特征输入映射为各因子的波动率，并对结果取绝对值以保证波动率的非负性。
