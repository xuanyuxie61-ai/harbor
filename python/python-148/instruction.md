## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目包含一个E-I（兴奋-抑制）神经质量模型的相关实现，缺失的代码主要由三个部分组成：

1. 第一部分是模型内部的动力学演化方法。该方法接收当前时间和状态变量，计算基于时间的锯齿波驱动信号。随后，需要分别计算兴奋性群体和抑制性群体的状态变化率。此处需要实现E-I振荡器的常微分方程组，变化率由线性衰减项、Sigmoid激活项以及外部驱动共同决定。计算激活项时需引入模型的各种连接权重、外部输入常数以及Sigmoid的阈值和宽度参数，最终返回由两者变化率组成的浮点数组。

2. 第二部分是一个计算雅可比矩阵的独立函数。该函数接收模型实例和平衡点坐标，用于评估系统在平衡点附近的稳定性。在计算时，假设平均周期驱动为零，先求出平衡点处的Sigmoid激活值，并根据激活值与其宽度参数计算出Sigmoid函数的导数值。接着，需要根据系统微分方程在平衡点求偏导的数学关系，将线性项系数、连接权重与Sigmoid导数相结合，构建并返回一个2x2的浮点型雅可比矩阵。

3. 第三部分是生成多通道局部场电位（LFP）信号的方法。该方法首先调用内部模拟数组的模拟接口，传入时间跨度和时间步长参数，以获取时间序列与状态演化数据。随后调用信号提取接口，传入兴奋与抑制的权重系数（值分别为1.0和1.2）、0.03的噪声标准差以及时间步长，将状态数据转换为多通道LFP信号，并将时间点与信号数据一同返回。

注意：项目中已有的数学工具函数（如Sigmoid激活函数、锯齿波函数等）及数组模拟与信号提取接口均已正确实现，可直接调用，无需重新定义。
