## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个结合了图神经网络与不确定性估计的分子属性预测模型，同时包含了物理约束机制。代码中缺失了三个关键部分的实现：

第一部分缺失了一个计算NIG（正态-逆伽马）分布负对数似然的类方法，用于回归任务的不确定性评估。该方法接收分布的四个参数以及真实标签值作为输入。在此处需要实现NIG负对数似然方程。算法思路为：首先计算一个中间变量，该变量由两倍的贝塔参数、努参数以及真实值与伽马参数之差的平方共同决定；随后，根据NIG方程计算各项对数似然损失，其中会用到对数函数和伽马对数函数；最终返回所有样本损失的平均值。

第二部分缺失了模型前向传播或预测阶段的结尾代码，负责整合特征并输出结果。算法思路为：首先获取全局特征，通过对节点特征在指定轴上进行求和池化，调整维度形状后输入全局多层感知机，并将输出展平；然后计算总能量，其值等于所有原子能量之和加上全局特征向量的第一个元素作为全局修正项；接着，将全局特征输入不确定性预测头，获取NIG分布的四个参数；最后，将原子能量、总能量、原子电荷、四个不确定性参数以及节点嵌入等结果以字典形式返回。

第三部分缺失了计算总训练损失的代码。算法思路为：首先调用不确定性预测头的NIG负对数似然方法，传入模型输出的四个不确定性参数及目标真实值，计算不确定性损失；然后计算电荷守恒物理损失，即预测的原子电荷总和与真实的原子序数总和之差的平方；最终，将均方误差损失、权重为零点一的不确定性损失，以及权重为给定物理约束系数的电荷损失相加，作为最终返回的总损失。
