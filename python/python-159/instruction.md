# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一部分缺失的是一个计算纵向声学模态的方法。该方法需要基于一端封闭一端开放（一端高阻抗近似速度节点与压力腹点，另一端低阻抗近似压力节点）的声学边界条件进行计算。需要遍历从1到指定阶数的模态，对于每个模态，利用声速和燃烧室长度，根据一端封闭一端开放的声学频率公式计算频率；接着计算对应的波长；并定义一个反映该边界条件的空间分布函数（提示：使用余弦函数）来表示模态振型。最后将模态信息列表和频率数组组合成字典返回。

第二部分缺失的是一个估计极限环振幅（即稳态振荡幅度）的方法，适用于Van der Pol型振荡器模型。首先需要判断系统的有效阻尼系数：若大于等于0，说明系统稳定，返回0；若小于0（不稳定），则需利用有效阻尼系数与非线性饱和参数来计算振幅。计算时需注意分母极小或非正的异常情况，此时应返回一个极大的饱和值（如1e6）；正常情况下，根据Van der Pol振子的极限环振幅方程计算并返回结果。

第三部分缺失的是一个执行热声耦合振荡器预测的流程函数。该函数首先需要实例化一个单模态热声振荡器，需从传入的分析器对象中获取第一阶纵向模态的频率作为自然频率，并设定声学阻尼为80.0、火焰增益系数为120.0、非线性饱和参数为5e7、初始压力扰动为200.0。接着，打印该振荡器的基频、有效阻尼、线性稳定性状态及极限环振幅。随后，使用四阶龙格库塔法在0至0.04秒的时间跨度内（步数20000）进行积分，并计算振荡指标（如峰峰值、均方根、频率、增长率等）进行打印。之后，构建一个多模态热声系统，提取分析器前3阶频率，阻尼数组设为[80.0, 200.0, 350.0]，在0至0.03秒内（步数15000）积分，并计算打印前3阶模态在时序末尾（如最后500个点）的最大绝对值振幅。最后返回单模态和多模态系统对象。
