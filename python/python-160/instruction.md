本项目涉及气化反应的热力学平衡计算。缺失的代码包含三个部分：一个基于牛顿法求解平衡组成的主方法、一个用于计算残差的辅助方法，以及一段调用求解器并进行统计检验的测试逻辑。

第一部分：主求解方法
功能：使用牛顿法求解气化反应的整体平衡组成。
算法思路：该方法接收温度、压力以及碳、水、氧的进料量作为参数。首先根据进料量构造元素守恒向量的右端项，并为五种主要气体组分设定一组正数的初始猜测值。在迭代循环中，计算当前各组分的总物质的量，并结合系统已有的方法获取特定温度下的两个平衡常数来构建残差向量。残差向量包含三个元素守恒方程的残差以及两个化学平衡约束方程（水煤气变换反应平衡和蒸汽重整反应平衡）的残差。随后，通过对变量施加微小扰动来计算数值雅可比矩阵，并求解线性方程组得到更新方向；若矩阵求解出现线性代数错误，则退回使用最小二乘法求解。为了保障迭代收敛，需引入线搜索阻尼机制，在每次迭代中逐步缩减步长，确保新解的残差范数小于旧解。更新变量时需将其限制在极小正数之上，直到残差范数小于设定的收敛阈值或达到最大迭代次数。
数值与参数：元素守恒向量基于C、H、O的进料量计算（注意H和O的系数需对应其分子式中的原子数）；迭代最大次数为一百次；数值微分的扰动常数取一千万分之一；收敛阈值取一亿分之一；变量下限截断值及总物质的量下限取极小正数。

第二部分：残差辅助方法
功能：计算并返回给定组分向量下的残差。
算法思路：作为主求解方法的辅助函数，接收组分向量、元素守恒向量和温度作为参数。其核心逻辑与主方法中的残差计算完全一致，即计算总物质的量，调用已有的方法获取相同的两个平衡常数，并构建包含三个元素守恒残差和两个化学平衡残差的五维向量。同样，在涉及除法的化学平衡残差计算中，分母需使用极小值进行截断保护以防止除零错误。

第三部分：调用与测试逻辑
功能：执行平衡组成计算，输出摩尔分数，并进行卡方检验。
算法思路：调用上述牛顿法求解器，传入特定的温度（一千零七十三开尔文）、压力（一百零一千三百二十五帕斯卡）和设定的进料条件。获取计算结果后，按顺序对应五种气体组分的名称，计算总摩尔数并打印各组分的摩尔分数（注意分母需做极小值保护）。随后，定义一组包含五个观测值的数组，将计算得到的平衡摩尔分数作为期望值，调用项目中已有的卡方检验函数计算并输出卡方值与p值。
