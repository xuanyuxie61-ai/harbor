## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个热流耦合的数值模拟系统。缺失的代码主要涉及三个关键部分的逻辑：

1. 热扩散率计算函数：该函数接收有效热导率、有效密度和有效比热容三个参数。在计算时，需先验证分母（即有效密度与有效比热容的乘积）是否大于零，若存在小于等于零的情况则抛出数值异常，最后返回热扩散率的计算结果。此处需要实现热扩散率的物理定义方程。

2. 达西通量计算与热求解器初始化：首先需要初始化水平方向的达西通量数组。在垂直方向的循环中，根据初始压力与生产压力之差以及水平方向长度计算压力梯度，调用已有的流体粘度函数获取粘度，并利用基质渗透率计算各层的达西通量。此处需要实现简化的达西定律方程。随后，调用前述的热扩散率计算函数获取全场热扩散率，并以此初始化一个ETDRK4热求解器实例，传入网格参数、区域长度、缩小为原十分之一的时间步长、热扩散率场、达西通量以及注入温度等参数。

3. 温度场演化中的扩散项与对流项计算：此部分位于求解器的迭代逻辑中。对于扩散项，需分别计算温度场在水平和垂直方向的二阶空间导数，采用中心差分格式，边界处导数置零，再结合热扩散率计算得到扩散贡献；对于对流项，需计算水平方向的对流，采用迎风格式，根据通量的正负情况分别选择左偏或右偏的一阶差分来计算温度梯度。此处需要实现热传导方程中的扩散项与对流项，其中扩散采用中心差分法，对流采用迎风格式。
