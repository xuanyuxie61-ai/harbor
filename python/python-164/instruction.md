# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

首先，缺失的第一部分需要实现一个带阻尼和边界保护的牛顿迭代算法，用于求解某个物理量（如过电位）。在最大迭代次数内，计算目标函数及其导数值。若导数绝对值极小（如小于1e-30）则提前终止迭代。迭代步长需通过阻尼机制限制（绝对值不超过0.1），并将更新后的变量限制在负2.0到正2.0的范围内。当相邻两次迭代的变量差值小于容差时退出循环。若最终结果非有限数，则将其重置为0.0并返回。

其次，缺失的第二部分需要实现一维稳态扩散-反应方程的有限差分离散化，并构建对应的三对角线性方程组。需定义长度为N-1的下对角和上对角向量、长度为N的主对角向量和右端项向量。内部节点的离散系数由有效扩散系数与网格间距平方的比值决定，主对角还需叠加反应速率常数。对于左边界，施加Dirichlet条件，将主对角设为1，上对角设为0，右端项设为左边界给定浓度；对于右边界，施加Neumann条件（浓度梯度为零），将主对角设为1，下对角末尾设为负1，右端项设为0。最后，调用项目中已有的Thomas算法函数求解该方程组得到浓度分布。

最后，缺失的第三部分需要根据电池运行参数计算反应速率常数。该常数与绝对电流密度成正比，与法拉第常数和氧气体相浓度的乘积成反比（为防止除零，氧气浓度需设置1e-6的下限），并取计算结果与下限0.1中的较大值。设定催化层厚度为10微米，网格数为51，左边界浓度等于氧气体相浓度。随后，分别调用项目中已有的三对角矩阵求解函数和带状矩阵求解函数计算浓度分布，并求取两种求解器结果的最大绝对差值，以验证求解器的一致性。
