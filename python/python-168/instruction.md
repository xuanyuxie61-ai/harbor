本项目实现了一个基于SE(2)位姿图的优化系统，涉及误差与雅可比计算、位姿的矩阵转换与相对变换求解，以及位姿图中各种边的构建。

缺失部分一：位姿图优化中计算SE(2)误差与雅可比的静态方法。该方法接收两个位姿变量和一个观测相对变换作为输入。算法思路为：首先根据两个位姿计算它们之间的预测相对变换，接着求取观测相对变换的逆变换，将预测变换与观测逆变换相乘得到误差变换，并从中提取出平移与旋转误差向量，其中旋转角需归一化到负π到正π的范围内。随后，计算该误差分别对两个输入位姿的雅可比矩阵。此处需要实现SE(2)群上的位姿复合与求导相关方程。

缺失部分二：实现SE(2)位姿与齐次变换矩阵之间的转换及相对变换计算。包含三个方法：第一个方法将当前对象的内部位姿转换为二维齐次变换矩阵；第二个静态方法将二维齐次变换矩阵反向提取为位姿向量；第三个方法计算从当前位姿到另一位置姿的相对变换，通过对当前位姿矩阵求逆并与另一位置姿矩阵相乘实现。此处需要实现二维刚体变换矩阵的构建与求逆运算。

缺失部分三：在位姿图中添加观测边。包含三部分逻辑：第一部分是添加里程计边，遍历连续关键帧，设置里程计信息矩阵（平移分量信息值为10，旋转分量信息值为50），利用机器人类实例计算两帧间的相对变换后添加边；第二部分是添加自动检测的环路闭合边，筛选属于关键帧集合且索引间隔大于1的闭环，设置闭环信息矩阵（平移分量信息值为5，旋转分量信息值为20），将检测到的变换作为边添加；第三部分是手动添加起点与终点的环路闭合以保证轨迹闭合，当关键帧数量不少于4时，使用真实位姿计算首尾关键帧的相对变换，并添加均值为0、标准差为0.02的高斯噪声来模拟观测不确定性，对旋转角进行归一化处理后添加边，并打印手动添加闭环的提示信息。
