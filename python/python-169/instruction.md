该项目涉及基于谱方法的轨迹优化与数值求解。缺失的代码分为三个部分：

第一部分：实现基于插值节点的微分矩阵构建算法。首先需要根据节点坐标向量计算重心插值权重，该权重由当前节点与其他所有节点坐标差值的连乘求倒数得到；随后利用这些权重和节点差值构建一阶微分矩阵，其中非对角线元素由特定节点权重比值与对应节点差值决定，而对角线元素需通过对该行所有非对角线元素求和并取负值进行赋值，以确保矩阵行和为零的性质。此处需要实现拉格朗日插值微分矩阵的构建逻辑。

第二部分：实现多自由度系统状态轨迹的细化修正。对于每个自由度，根据其起始状态、目标状态及时间区间跨度计算恒定的期望关节速度；接着构建一个维度与节点数相同、元素全为该期望速度的右端向量；最后，调用项目内已有的稀疏线性代数求解器，对由缩放微分矩阵和该右端向量构成的线性系统进行求解，以获取满足期望速度约束的修正轨迹。此处需要实现期望速度映射及基于微分矩阵的线性方程构建与求解。

第三部分：实现基于正规方程的共轭梯度迭代求解器（如CGNE算法）。算法需兼容密集矩阵和稀疏算子两种输入格式，在每次迭代中，通过计算矩阵与搜索方向的乘积、步长以及伪残差的范数来不断更新解向量、残差向量和搜索方向。当伪残差的范数小于给定的收敛容差时终止迭代；若迭代过程中解向量出现非有限数值，则将解重置为零向量并提前退出。此处需要实现共轭梯度法在正规方程上的迭代更新逻辑。
