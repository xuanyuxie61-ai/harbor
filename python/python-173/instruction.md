本项目实现了一个基于有限元方法的对流-扩散-反应方程的数值求解器。缺失的代码主要分布在三个核心部分：

第一部分缺失代码位于局部刚度矩阵的组装过程中。此部分需要计算三角形单元上的局部刚度矩阵贡献。算法思路是遍历单元的三个节点，计算形函数梯度与扩散系数的点积，以及形函数自身与反应系数的乘积，将这两部分相加后，乘以积分权重和雅可比行列式，累加到局部矩阵中。此处需要实现有限元方法中扩散-反应项的局部组装逻辑。

第二部分缺失代码位于时间步进函数内部，负责根据不同的时间离散格式构造线性方程组的左端项和右端项。算法思路需要根据传入的格式参数进行分支处理：当格式为隐式时，左端项由集中质量矩阵、时间步长与扩散及对流矩阵之和构成，右端项由质量矩阵与当前解的乘积加上时间步长与载荷向量的乘积构成；当格式为Crank-Nicolson时，需要按照该格式特定的半时间步长系数来组合质量矩阵、扩散对流矩阵及载荷向量，构造对应的左右端项；若格式未知则抛出异常。此处需要实现抛物型偏微分方程的隐式和Crank-Nicolson时间离散格式。

第三部分缺失代码位于主求解过程中，负责时间步的推进。算法思路是通过一个指定步数的迭代循环，在每一时间步调用时间步进函数来更新解向量。调用时需要传入网格节点与单元信息、当前解、时间步长、各种物理参数函数（扩散、反应、对流、源项等）、狄利克雷边界条件信息以及集中质量矩阵，并默认使用隐式格式。此处需要实现时间演化过程的主循环调用逻辑。
