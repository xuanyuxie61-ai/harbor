## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个基于树结构的多极展开算法（如快速多极子方法）的实现，用于高效计算带电粒子间的势场。代码中存在三处缺失，分别对应多极矩的直接计算、多极矩的向上传递合并以及远场势的评估。

第一处缺失位于多极矩直接计算的函数中。该函数的功能是将粒子的电荷累加到多极矩中。算法思路为：遍历所有粒子，计算粒子到展开中心的距离。若距离极小（小于一个极小阈值，例如10的负15次方，可视为在原点处），则仅保留零阶项的贡献，将该粒子电荷累加到零阶矩及总电荷中并跳过后续计算；否则，将该粒子电荷累加到总电荷，并针对从零到展开阶数的每一阶，计算距离的阶数次幂，同时利用归一化缔合勒让德多项式计算球谐函数的值。对于磁量子数为零的项，仅计算实部；对于磁量子数不为零的项，需结合方位角的整数倍正弦与余弦值，分别计算并累加实部与虚部。此处需要实现基于球谐函数展开的多极矩计算方程。

第二处缺失位于自底向上计算多极矩的函数中。该函数的功能是实现树结构中多极矩的层次化构建。算法思路为：首先获取所有叶子节点，对含有粒子的叶子节点直接实例化多极展开对象并计算其多极矩（即P2M过程）；随后通过递归向上遍历非叶子节点，初始化当前节点的多极矩实部与虚部数组，遍历其子节点，对于含有数据的子节点，调用已存在的多极矩平移算子函数，将子节点的多极矩平移至当前节点中心并累加（即M2M过程）。此处需要实现多极矩到多极矩的合并逻辑。

第三处缺失位于目标点势能评估的函数中。该函数的功能是计算远场粒子对目标点的势贡献。算法思路为：在遍历所有叶子节点时，跳过邻居叶子节点和空叶子节点，对于剩余的远场叶子节点，实例化多极展开对象，将远场叶子节点已计算好的多极矩实部与虚部赋值给该对象，并调用其势能评估方法计算该远场对目标点的势，将结果累加。此处需要实现基于多极展开的远场势评估逻辑。
