你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个结合了一维有限元方法与随机谱方法（如多项式混沌展开）的数值求解程序，主要用于求解带有随机参数的扩散-反应方程。缺失的代码分布在三个关键步骤中：

1. 第一个缺失片段对应有限元局部单元矩阵的计算。你需要实现扩散项对应的局部刚度矩阵以及反应项对应的局部质量矩阵。其中，局部刚度矩阵的构建依赖于平均系数与单元长度的比值，其元素呈现正负交错的结构；局部质量矩阵依赖于单元长度与特定比例常数的乘积，其主对角线与副对角线元素的值呈现二比一的比例关系。

2. 第二个缺失片段对应局部矩阵向全局矩阵的组装。在此步骤中，需要先将原有的平均系数折半（这可能对应某种半隐式时间推进格式或特定的数值平均策略），随后再次计算局部刚度矩阵。接着，依据当前单元及其相邻节点的全局索引，利用双重循环将局部刚度矩阵中的各个元素累加至全局刚度矩阵的对应位置。

3. 第三个缺失片段对应随机输出方差的计算。在随机伽辽金或谱展开的框架下，你需要通过遍历多指标集来求解方差。计算规则为：排除零阶多指标（该指标对应随机变量的均值项），将剩余所有非零阶指标对应的展开系数求平方后进行累加。注意方差的初始值应设为零。

请根据以上算法思路与功能描述，在项目中定位缺失位置并完成代码补全。
