## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失片段负责有限元全局矩阵的组装。算法上，需要计算三角形单元的局部质量矩阵和局部刚度矩阵：局部质量矩阵的值由单元面积除以12再乘以一个由2和1组成的系数矩阵得到；局部刚度矩阵的值由4倍单元面积的倒数乘以梯度相关向量的内积得到。计算完成后需将其组装至全局质量与刚度矩阵中。接下来，需要通过统计边被单元共享的次数来识别边界边，并对这些边界边计算一维线单元的局部边界质量矩阵，其值由边界长度除以6再乘以由2和1组成的系数矩阵得到，随后乘以给定的边界参数组装到全局边界控制矩阵中。此处需要实现标准二维三角形网格的有限元空间离散及含参数的边界弱形式积分项组装。

第二个缺失片段负责非线性状态方程的时间正向推进求解。算法思路为半隐式时间离散与非线性项线性化：利用上一时间步的状态变量对三次方非线性项进行近似，线性化系数为3.0。为保证数值稳定性，需采用质量集中方法（即将全局质量矩阵每行求和提取为对角矩阵）来处理线性化项的对角贡献。构建左端矩阵时，需在质量矩阵的基础上叠加上时间步长乘以扩散项矩阵，以及时间步长乘以非线性项系数、状态变量平方和集中质量的对角贡献；右端向量则由质量矩阵与上一时间步状态的乘积，加上时间步长乘以外力与控制项构成。最后求解该线性系统更新状态。此处需要实现含非线性反应项和扩散项的演化方程的半隐式时间步进及线性化求解。

第三个缺失片段负责最优控制中伴随方程的时间反向推进求解。算法上需在逆时间循环中计算当前步的伴随变量：右端向量由下一时间步的伴随变量与质量矩阵的乘积，加上时间步长乘以质量矩阵与状态变量、目标状态变量偏差的乘积构成。左端矩阵的构造与状态方程的线性化算子保持一致，同样需要使用集中质量矩阵，并叠加上系数为3.0的非线性项线性化对角贡献，此时线性化需使用当前时间步对应的状态变量值。此处需要实现伴随方程的反向时间步进求解。
