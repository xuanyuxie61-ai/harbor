## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

项目中包含一个水平集方法相关的类与时间推进逻辑，缺失了三个关键部分的实现：

1. 时间积分步进函数（缺失部分1）：需要实现一个三阶强稳定性保持龙格-库塔方法（SSP-RK3）。该函数接收当前状态变量、时间步长和右端项计算函数，分三步完成状态更新。第一步利用初始状态和第一步右端项计算中间状态；第二步利用四分之三的初始状态、四分之一的中间状态以及四分之一时间步长乘以第二步右端项，计算第二步中间状态；第三步利用三分之一的初始状态、三分之二的第二步中间状态以及三分之二时间步长乘以第三步右端项，计算并返回最终的新状态。

2. 组合右端项计算函数（缺失部分2）：需要实现水平集演化的组合右端项计算。方向提示：此处需要实现包含曲率项、对流项和外部法向强迫项的演化方程右端项。算法思路为：首先临时创建一个同类型的水平集对象来辅助计算当前场的曲率和梯度范数，接着调用已有的二阶中心差分函数计算状态变量的空间偏导数，然后将三部分按物理意义组合（其中曲率项乘以一个给定的极小常数，对流项由速度场分量与对应偏导数相乘后相减构成，强迫项由外部速度场与梯度范数相乘构成）。边界处理采用零梯度条件，将四个边界行或列的值分别设为紧邻的内部行或列的值。

3. 界面平均曲率计算函数（缺失部分3）：需要实现水平集界面的平均曲率计算。方向提示：此处需要实现水平集曲率公式（即梯度的散度形式）。算法思路为：调用已有的二阶中心差分函数计算一阶偏导，然后在内部网格通过差分格式计算三个二阶偏导，并在边界处采用零梯度延拓（将边界值赋为相邻内部值）。分母为梯度平方的一点五次方，为防止除零，需对梯度平方设置一个极小下限（如1e-12）进行保护。为保障数值稳定性，需对最终计算出的曲率进行上下限截断处理，限制在正负一千之间。
