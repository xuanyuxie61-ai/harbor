第一个片段是一个数值通量计算函数。该函数接收左右单元的守恒变量状态和面法向量。首先，调用已有的函数分别计算左右状态在法向的物理通量，并将守恒变量转换为原始变量以提取速度分量，同时计算各自的声速。接着，计算左右状态沿法向的速度绝对值，并将其分别与各自声速相加，取两者中的最大值作为最大波速。最后，根据Rusanov（局部Lax-Friedrichs）格式的思想，将左右物理通量的平均值减去基于最大波速的一半与左右状态差值的乘积作为耗散项，返回最终的数值通量。

第二个片段是针对可压缩流体方程（如Euler方程）的表面积分部分，用于组装右端项。代码需遍历每个四面体单元及其四个面。对于每个面，获取法向量和邻居单元索引。在面的积分点上，计算当前单元的解作为左状态。如果存在邻居单元，则获取邻居单元对应面上的解作为右状态；如果是边界，则根据是否存在给定的边界函数来计算右状态，若未给定，则默认采用反射壁面边界条件（即保持密度和压力不变，将法向速度分量进行镜面反射，具体做法是从原速度各分量中减去两倍的法向速度分量与对应法向分量的乘积，再重构为守恒变量）。接着，根据通量类型选择调用已有的Roe或Rusanov通量函数计算数值通量。最后，将数值通量与物理权重、面基函数进行外积运算，累加到对应单元的右端项中，注意累加时符号为负。

第三个片段是针对线性对流方程的表面积分部分，用于组装右端项。代码同样需遍历单元和面，获取法向量，并计算给定的常对流速度在法向上的投影。在面的积分点上，获取左状态，并根据邻居单元或边界条件获取右状态（若存在边界函数则调用，否则默认采用零梯度出流条件，即右状态等于左状态）。接着，根据对流速度法向投影的符号进行上风格式选择：若投影非负则取左状态，否则取右状态作为界面数值通量。最后，将数值通量与物理权重、法向对流速度投影以及面基函数相乘，累加到右端项中，符号同样为负。
