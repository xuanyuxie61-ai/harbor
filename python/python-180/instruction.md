## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及随机微分方程（SDE）的数值求解，特别是针对带有空间承载力与噪声项的动力学模型。需要补全的代码包含三个部分：

1. 缺失的第一个部分是一个独立的SDE单步推进函数，实现了Milstein强阶1.0方法（针对单维噪声）。该算法的思路是在常规欧拉离散化的基础上，利用漂移项、扩散项以及扩散项对状态的导数进行状态更新。函数需接收当前状态、漂移函数、扩散函数、扩散导数函数、步长和维纳过程增量作为参数，并在计算前校验步长必须为正值。此处需要实现Milstein方法的迭代方程，该方程除了包含漂移与扩散的常规线性项外，还需加上一个由扩散项值、其导数值、维纳增量平方与步长共同决定的修正项。

2. 缺失的第二个部分是一个类方法，用于计算扩散系数的对角Jacobian（即导数）。在该模型中，扩散项受到噪声强度参数和空间承载力参数的影响。此方法的算法思路是求取扩散项对状态的导数，其值为噪声强度参数乘以一个关于状态与承载力比例的线性表达式，该线性表达式的常数项为1，而状态变量项的系数为承载力倒数的负两倍。

3. 缺失的第三个部分是一个条件分支逻辑，当积分器的方法被设定为“milstein”时执行。该部分的作用是调用上述的Milstein单步推进函数，并按顺序传入当前状态变量、类自身的漂移计算方法、扩散计算方法、扩散Jacobian计算方法，以及时间步长和维纳增量。

请注意，项目中已有的其他函数和方法（如漂移函数、扩散函数等）默认已正确实现，不需要重新定义，直接调用即可。
