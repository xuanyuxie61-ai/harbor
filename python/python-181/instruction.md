你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目旨在实现基于非线性薛定谔方程的谱嵌入算法。缺失的代码主要包含以下三个部分：

第一部分：构建有效势能的函数。该部分需要根据输入的数据点和尺度参数，计算数据点之间的距离，基于高斯核密度估计实现势能构建。具体思路为：对每个数据点，计算它到所有点的距离并利用高斯核计算局部密度，对密度取负对数（注意需加一个极小值以防止对数计算溢出），最后对结果减去最小值进行归一化处理并返回势能数组。

第二部分：计算非线性薛定谔方程总能量的函数。该部分需要接收波函数、图拉普拉斯矩阵、势能和非线性系数作为输入，实现非线性薛定谔方程的能量泛函计算。总能量由三部分组成：基于图拉普拉斯矩阵的动能、基于势能函数的势能项，以及与非线性系数和波函数模的四次方相关的非线性相互作用能，最后返回这三部分的总和。

第三部分：非线性薛定谔方程谱嵌入的主流程模块。该模块负责整合计算流程并输出结果。具体而言，需要依次调用已实现的线性谱嵌入函数和非线性谱坐标计算函数，其中维度参数设定为3，尺度参数设定为1.5，非线性系数设定为-0.5，非线性迭代次数设定为30。随后，调用第一部分的函数构建有效势能（尺度参数同样设定为1.5），并基于非线性嵌入结果的第一列和图拉普拉斯矩阵计算基态能量。最后，调用已有函数计算非线性嵌入的等距保持质量，打印相关的特征值、能量值和质量值，并将线性嵌入、非线性嵌入、特征值和能量结果以字典形式返回。
