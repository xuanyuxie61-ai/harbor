你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目涉及图模型中精度矩阵的估计与统计假设检验。缺失的代码主要由三个功能部分组成：

1. 第一部分实现了Graphical Lasso算法的迭代求解过程。该部分需要在一个最大迭代次数的循环中，计算当前精度矩阵的逆矩阵；若遇到矩阵奇异错误，需加入微小扰动（如1e-6乘以单位矩阵）以保持正定性。接着计算梯度，即逆矩阵与样本协方差矩阵的差，并执行梯度下降；使用已有的软阈值函数进行L1正则化，其阈值由学习率和正则化系数决定。随后通过特征值分解将矩阵投影到正定锥，将小于极小正值（如1e-8）的特征值截断以保证正定性。计算本次迭代与上次迭代矩阵在Frobenius范数下的差异，若开启详细输出且迭代次数为20的倍数，则计算并打印目标函数值（该值由对数行列式、矩阵迹及L1范数惩罚项组成）；当差异小于容差时提前终止循环，最后返回更新后的精度矩阵。

2. 第二部分实现了基于精度矩阵的偏相关性检验。该部分需要遍历矩阵的上三角区域，利用对角线和非对角线元素计算偏相关系数，并将其裁剪至接近负一到正一的区间（如-0.9999到0.9999）以防止极端值。接着根据偏相关系数和自由度（样本数减维度减2，且最小为1）计算t统计量；然后利用误差函数进行正态近似，计算双边p值；最后将p值与给定的显著性水平比较，判断是否拒绝原假设，并填充对称的p值矩阵和拒绝矩阵，最后返回这两个矩阵。

3. 第三部分实现了合成数据的生成与检验流程。首先获取稀疏精度矩阵的维度，在原矩阵上加上0.5倍单位矩阵后求逆，得到协方差矩阵；从均值为零向量的多元正态分布中采样生成数据；计算样本协方差矩阵，对其加上0.1倍单位矩阵后求逆，得到估计的精度矩阵；调用已有的偏相关性检验函数（显著性水平设为0.05），并根据返回的拒绝矩阵统计显著边的数量（即拒绝矩阵中为真的数量除以2）。
