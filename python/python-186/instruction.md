# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

这是一个基于扩展SEIR仓室模型的传染病动力学模拟与参数校准项目，模型包含易感、潜伏、无症状感染、有症状感染、住院、康复和死亡等七个状态。

缺失的代码包含三个部分，具体功能与算法思路如下：

第一部分：计算模型各状态随时间变化的导数（即微分方程组的右端项）。首先，需要从参数字典中提取人群总数、传播率、无症状相对传染率、潜伏期转化率、有症状比例、各阶段恢复率、住院率、死亡率以及免疫丧失率等参数。接着，对各个状态变量进行非负边界保护。然后，计算总传染力与感染力，并依据传染病动力学方程计算各仓室的变化率。完成变化率计算后，需要进行守恒性检查：若总人数的变化量超过极小阈值，则需根据当前总人数与各状态总和的比值对各项变化率进行归一化修正。最后，将各导数以浮点数组的形式返回。

第二部分：计算该传染病模型的基本再生数（R0）。需要从参数字典中获取传播率、有症状比例、无症状相对传染率、各恢复率及住院率等参数。算法思路是将基本再生数拆分为有症状和无症状两部分分别进行解析计算，注意分母需做极小值保护以防除零，最后将两部分的贡献相加并返回。

第三部分：根据给定的目标基本再生数反向校准传播率参数。首先，定义一个内部函数，该函数能够根据传入的传播率值及参数模板，按照第二部分的解析思路计算当前的基本再生数；再定义一个误差函数，表示当前再生数与目标值的差值。接着，设定传播率的初始搜索区间，通过循环动态扩展该区间，直到误差函数在区间两端点的值符号相反（即存在零点）。最后，调用项目中已正确实现的二分法求根函数求解并返回校准后的传播率。
