# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一处缺失代码需要实现基于Jacobi迭代的偏微分方程求解过程。算法思路是通过迭代更新解向量，在每次迭代中调用已有的拉普拉斯算子计算函数，结合右端项以及给定的参数（包含系数alpha和时间步长dt）进行更新，同时需要保证已知掩码区域的值强制保持为给定矩阵的对应值。当两次迭代结果的误差范数小于给定的收敛阈值（1e-5）或达到最大迭代次数（50次）时，停止迭代。此处需要实现隐式扩散方程的迭代求解形式。

第二处缺失代码需要实现计算点在四面体中的重心坐标。算法思路是利用点与四面体顶点构成的子四面体的有向体积与总体积的比值来计算前三个坐标分量，第四个坐标分量则利用坐标分量之和为1的性质计算得出。最后将四个分量组合成数组返回。此处需要实现基于体积比的重心坐标计算。

第三处缺失代码需要实现多个预测结果的加权融合与评分限制。算法思路是对四个不同来源的预测矩阵（扩散、有限元、时间以及相似度核）进行加权求和。给定的四个权重依次为0.35、0.25、0.25和0.15。注意，相似度核矩阵在加权前需要先进行线性变换，即乘以缩放因子3.0并加上偏置2.5，且只需截取指定用户数和物品数的子矩阵参与计算。最后，需要将融合后的预测结果裁剪限制在1.0至5.0的闭区间范围内。此处需要实现多模型加权融合与评分截断。
