# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

第一个缺失部分是一个计算策略参数对数概率梯度的方法。该方法需要分别计算均值参数和对数标准差参数的梯度。算法思路上，首先需要利用已有的方法获取状态的基函数评估结果、策略均值、标准差，并计算动作与均值的差值。对于均值参数的梯度，此处需要实现策略梯度的对数概率方程中关于均值的解析导数，其计算逻辑涉及利用差值与方差的比值以及基函数，最终呈现为两者的外积形式。对于对数标准差的梯度，此处需要实现策略梯度的对数概率方程中关于对数标准差的解析导数，计算逻辑为差值的平方与方差之比，再减去一个数值为1的常数，以表示归一化平方差与1的偏差。最终将两部分梯度以字典结构返回。

第二个缺失部分是一个获取展平策略梯度的方法。算法思路是调用策略对象已有的梯度计算方法获取梯度字典，提取出均值和对数标准差对应的梯度后，将它们分别展平为一维数组，然后按顺序拼接成一个完整的一维向量并返回。

第三个缺失部分是一个计算Fisher向量积的函数，目的是在不显式构造Fisher信息矩阵的情况下计算矩阵与向量的乘积。算法思路是遍历输入的状态和动作对，通过传入的策略梯度计算函数获取每一对的策略梯度向量，此处需要实现Fisher向量积方程，即计算该梯度与输入向量的内积，再将该梯度乘以内积结果进行累加，最后对所有累加结果求平均。如果输入的状态动作对数量为零，则直接返回与输入向量形状一致的全零向量。
