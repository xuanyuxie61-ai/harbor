本项目结合了流体力学数值计算与物理信息生成模型。缺失的代码分为两个部分：

第一部分的功能是计算三维不可压缩流体动量方程的残差。算法思路为：分别针对X、Y、Z三个空间方向，计算流场的对流项、压力梯度项和粘性扩散项。其中，压力项需要乘以密度的倒数（密度倒数取值为1.0除以给定的密度变量），粘性项需要乘以运动粘度系数。动量残差由对流项与压力项之和减去粘性项构成。此处需要实现Navier-Stokes方程的稳态残差计算（忽略时间项），假设已有计算空间偏导数和二阶偏导数的函数可直接调用。最终，需要将各方向动量残差的平方均值、已有的连续性残差以及它们的总和以字典形式返回。

第二部分的功能是在生成模型的训练循环中周期性地计算并反向传播物理损失。算法思路为：初始化二元交叉熵和均方误差损失函数。为了降低计算开销，每5个训练轮次，且在批次索引为0以及样本数等于完整网格总节点数时，执行物理损失计算。具体地，利用Box-Muller变换生成指定维度的隐向量，将其形状重塑并复制扩展至全网格大小；随后将扩展后的隐向量与坐标信息输入生成器，得到生成的物理场；接着，将生成场与真实状态场中从第5列开始的数据进行均方误差计算，作为物理损失的代理；最后，计算该损失的梯度，乘以物理损失权重系数后反向传播给生成器，并将该损失值累加至当前轮次的物理损失记录中；若不满足上述执行条件，则当前物理损失记为0。
