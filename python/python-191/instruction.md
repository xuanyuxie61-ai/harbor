# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

片段一的功能是生成圆盘积分所需的极坐标网格和权重。算法思路为：角度变量需在0到2倍的圆周率区间内，根据角度方向的离散点数进行均匀离散化；半径需要对径向输入变量进行0到1的截断处理后，应用平方根变换以实现圆盘面积微元的均匀分布；权重则由给定的径向权重除以角度离散点数得到。该片段最后需返回权重、半径和角度。

片段二的功能是在单位圆盘上生成均匀分布的随机采样点。算法思路为：首先在0到2倍的圆周率区间内生成均匀随机数作为角度，然后在0到1区间内生成均匀随机数并取其平方根作为半径，此平方根变换同样是为了保证在圆盘面积上的均匀性。接着，需利用三角函数将极坐标转换为笛卡尔坐标，最后将横纵坐标按列拼接并返回。

片段三的功能是验证圆盘积分函数的正确性。算法思路为：首先定义一个被积核函数，该函数需计算输入的笛卡尔坐标分量的平方和，即对应极径的平方。随后，调用项目中已有的圆盘积分函数，并传入该核函数，同时将径向和角度方向的离散点数分别设为16和32。接着，计算该核函数在单位圆盘上的解析积分真实值，其值为圆周率的一半。最后，将计算得到的积分值与真实值进行格式化对比打印输出。
