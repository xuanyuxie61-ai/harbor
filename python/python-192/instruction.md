## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个涉及计算流体力学与偏微分方程数值求解的程序，主要包含高阶数值格式的重构限制以及泊松方程的频域求解。缺失的代码由三个功能模块组成：

第一个片段是一个用于抑制数值振荡的斜率或通量限制器函数的返回语句。该函数接收两个参数：一个是相邻网格梯度的比值，另一个是控制限制器陡峭程度的参数。其算法思路是确保返回值具有非负性，并在由该陡峭参数与梯度比值的乘积、一加梯度比值的一半，以及该陡峭参数本身三者之间取最小值，从而实现总变差递减（TVD）的性质。

第二个片段是一个用于MUSCL（Monotonic Upstream-centered Scheme for Conservation Laws）高阶重构的函数，目的是在给定的空间轴方向上计算界面的左右状态。算法思路为：首先根据方向对守恒变量场进行切片，提取界面两侧的左右单元值；接着计算当前单元及相邻单元的正向与反向差分；随后利用项目中已存在的安全除法函数计算左右两侧的梯度比值；最后，调用项目中已实现的minmod限制器函数对梯度进行限制，左状态值需加上半倍的限制梯度与正向差分的乘积，右状态值需减去半倍的限制梯度与对应右侧正向差分的乘积，以此完成高阶线性重构。

第三个片段是利用谱方法在频域求解泊松方程的核心逻辑。算法思路为：在频域空间中初始化解的数组，对于零频分量（即直流分量），需将其值设为零以满足物理上的零均值约束；对于其他频率分量，需要计算离散拉普拉斯算子的特征值（修正波数），该特征值由x和y方向的网格间距以及对应离散波数的余弦函数计算得出；最后，将频域源项取负并除以该特征值，即可得到频域中的解。
