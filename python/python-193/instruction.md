# 任务：修复挖空代码（Multi-Hole Benchmark）

## 目标
你面对的是一组Python 的科研代码。代码中有多处被"挖空"（函数体、关键逻辑、边界条件等被删除或替换为占位符），导致代码无法正确运行或输出错误结果。

## 工作目录
代码仓库位于 `/app` 目录下。

## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目实现了一个二维反应扩散方程的有限元求解及多种预条件共轭梯度法（PCG）的求解与性能对比。缺失的代码主要分为三个部分：

第一部分：有限元全局刚度矩阵与载荷向量的组装。
- 功能：遍历所有三角形单元，计算局部刚度矩阵和载荷向量，并组装到全局矩阵和向量中。
- 算法思路：
  1. 提取单元节点索引，进行越界保护，将超出节点总数的索引限制在合法边界范围内。
  2. 根据参考单元到物理单元的仿射映射，计算雅可比矩阵、其行列式以及单元面积（行列式绝对值的一半）。若行列式绝对值过小（接近1e-15），则跳过该退化单元。
  3. 此处需要实现重心坐标对物理坐标的梯度计算：利用雅可比矩阵逆的转置与参考单元下的梯度矩阵相乘。
  4. 在单元质心处调用已实现的扩散和反应函数获取系数，计算扩散刚度矩阵和反应刚度矩阵。其中反应刚度矩阵使用一致质量矩阵的形式，对角线系数与面积和反应系数相关（系数值为面积除以6），非对角线系数值为面积除以12。将两者相加得到总单元刚度矩阵。
  5. 通过数值积分计算载荷向量：映射积分点到物理坐标，调用源项函数，结合积分点处的重心坐标、积分权重和雅可比行列式绝对值进行累加。
  6. 将单元矩阵和向量累加至全局刚度矩阵和全局载荷向量中，最后返回全局矩阵和向量。

第二部分：预条件共轭梯度法（PCG）的迭代求解过程。
- 功能：实现PCG算法的核心迭代循环，支持并行矩阵向量乘法和预条件子。
- 算法思路：
  1. 在最大迭代次数内循环，根据进程数选择调用并行或串行的矩阵与搜索方向的乘积。
  2. 检查是否发生数值崩溃（搜索方向与乘积的内积绝对值小于1e-15），若崩溃则退出循环。
  3. 计算步长（旧残差与预条件残差内积除以刚才的内积），并更新解向量和残差向量。
  4. 计算相对残差范数并记录历史。若残差小于给定容限，则返回解及包含迭代次数、残差、收敛状态和历史记录的字典。
  5. 若提供了预条件子则应用于残差，否则直接复制残差。
  6. 计算新的内积，检查旧内积是否接近零以防止除零，然后计算搜索方向的更新系数，更新搜索方向，并保存新的内积。
  7. 循环结束后若未收敛，返回解和标记为未收敛的信息字典。

第三部分：不同预条件子的求解对比与效率评估。
- 功能：使用不同的预条件子调用PCG求解器，对比迭代次数并计算加速比。
- 算法思路：
  1. 初始化全零向量作为初始解。
  2. 分别使用无预条件、Jacobi预条件、SSOR预条件以及多重网格（若可用）调用PCG求解器。求解容限设为1e-8，最大迭代次数设为维度两倍与1000中的较小值。
  3. 打印每种方法的迭代次数、残差和收敛状态。
  4. 使用已有的安全除法函数计算并打印Jacobi、SSOR以及多重网格预条件相对于无预条件的迭代次数加速比（保留两位小数）。
  5. 最终返回使用Jacobi预条件得到的解向量。

请注意，项目中涉及的外部函数（如扩散函数、反应函数、源项函数、并行矩阵向量乘法、安全除法等）均已正确实现，可直接调用。请根据以上提示，在代码中找到相应的缺失位置并完成补全。
