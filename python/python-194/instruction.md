## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

本项目是一个计算流体力学数值求解器，主要涉及瞬态Stokes方程的数值模拟。缺失的代码分为三个部分，分别对应解析源项计算、分步投影法时间推进和主控仿真流程。

缺失部分1：解析源项计算函数的尾部。
该部分需要根据已知的速度场和压力场的解析表达式，计算动量方程的右端源项。你需要先计算速度分量及其二阶空间导数，以及压力梯度的分量。具体来说，需要利用乘积法则对给定的多项式形式的速度分量求二阶导数，然后结合粘性系数和压力梯度，根据Stokes动量方程的微分形式计算出两个方向的源项分量，并返回满足无散度约束的第三个分量。

缺失部分2：瞬态Stokes分步投影法单步推进函数。
该部分需要实现分步投影法的一个完整时间步。算法思路如下：
第一步，构建隐式扩散步的线性系统矩阵，该矩阵由粘性项的离散矩阵和密度与时间步长之比构成的系数项组合而成，需使用带状对称正定矩阵结构存储，并采用带状Cholesky分解求解中间速度场。右端向量由上一时间步的速度、当前体积力及压力梯度项组装。
第二步，进行压力修正，计算中间速度场的散度，并求解简化的压力Poisson方程（此处近似为标量除法，需注意对散度算子范数平方进行下限保护以防除零）。
第三步，更新速度场和压力场，速度场需减去时间步长与密度之比乘以散度算子与压力修正量的乘积，压力场累加修正量，并计算用于Verlet追踪的辅助速度场。

缺失部分3：瞬态Stokes仿真演示主程序。
该部分需要设置初始条件并驱动时间步进循环。你需要初始化速度、辅助速度和压力场，并构建一阶散度差分算子（采用相邻节点差分并除以网格间距的格式）。在计算初始右端项后，进行固定次数的迭代调用单步推进函数，并输出最终时刻的速度范数和压力均值。
