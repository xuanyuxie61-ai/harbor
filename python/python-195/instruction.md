## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

这是一个基于有限元方法的数值计算项目，主要包含网格数据结构的转换、单元刚度矩阵的组装以及网格质量的校验。

缺失部分1：实现混合网格到纯三角形网格的转换逻辑。你需要遍历项目中的所有单元，根据单元类型进行分支处理：若为四边形单元，需按特定的对角线将其拆分为两个三角形，并分别记录其节点索引；若为三角形单元，则直接提取其节点列表。最后，将收集到的所有三角形节点索引整体偏移，转换为从1开始的索引体系，并与节点坐标数据一同返回。

缺失部分2：实现三角形单元局部刚度矩阵的组装与数据提取。在三节点三角形单元的双层局部节点循环中，利用单元面积与形函数对空间坐标的偏导数，根据有限元中拉普拉斯算子（或泊松方程）的弱形式方程，计算局部刚度矩阵中对应位置的元素值，并将该值及其对应的行列全局节点索引分别追加到用于构建稀疏矩阵的数据列表中。

缺失部分3：实现三角形网格的有效性验证与统计。遍历所有生成的三角形，获取其顶点坐标并调用已有的面积计算函数。将计算得到的面积与一个极小的正数阈值（1e-14，用于过滤退化或共线的无效单元）进行比较，若面积大于该阈值则视为有效，并在所有有效单元中追踪记录最小的面积值，最后将此最小面积以科学计数法格式化打印输出。
