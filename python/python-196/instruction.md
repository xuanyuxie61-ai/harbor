## 要求
你现在在一个项目中，里面缺失了一部分的代码，请你找到缺失的位置并补全。代码的简介如下：

这是一个基于二维三角形单元的有限元计算项目，主要涉及网格生成、网格质量评估以及有限元刚度矩阵与载荷向量的组装。

第一部分代码的功能是生成规则矩形区域上的三角形网格。算法思路为：根据网格划分的行列数计算总三角形单元数（其值为矩形个数的两倍），通过双重循环遍历网格内部矩形，将每个矩形划分为两个三角形，并为每个三角形的三个顶点赋予对应的全局节点编号，最终构建出完整的单元-节点拓扑关系数组。

第二部分代码的功能是计算三角形网格的单元质量因子。算法思路为：遍历所有三角形单元，提取顶点坐标并计算单元面积以及三条边的长度平方和。通过面积与边长平方和的比值来评估单元的几何质量。当边长平方和小于极小阈值时，质量因子直接赋零以避免除零错误；否则，计算比值并乘以一个特定的与根号3相关的归一化常数系数，使得理想正三角形的该指标值为一。

第三部分代码的功能是有限元全局刚度矩阵和右端载荷向量的组装。算法思路为：遍历每个三角形单元，提取节点坐标并计算面积，若面积小于极小阈值则跳过。采用中点积分法则，将每条边的中点作为求积点，积分权重设为三分之一。在每个求积点上，计算测试函数和试探函数在当前点处的值及其对空间坐标的偏导数。最后，将源项函数与测试函数值的乘积累加至全局载荷向量，同时根据给定的传导率参数以及偏导数的点积，将单元刚度贡献累加至全局刚度矩阵。此处需要实现稳态热传导方程或泊松方程的有限元离散，涉及形函数及其梯度在单元上的积分。项目中已有的其他函数（如源项函数）默认已正确实现，直接调用即可。
