本项目旨在实现基于多项式混沌展开（PCE）的随机Galerkin求解方法，并通过蒙特卡洛模拟对解析解进行验证。缺失的代码包含三个核心功能部分：

第一部分功能：构建PCE的随机Galerkin矩阵。
算法思路：根据给定的PCE阶数确定矩阵维度并初始化全零方阵。遍历矩阵行，利用已存在的Hermite多项式双重内积函数计算对角线基础值，并将其与随机变量的均值参数相乘进行赋值。若随机变量的标准差参数不为零，则进一步遍历矩阵列，利用已存在的Hermite多项式三重内积函数计算交叉影响，将其与标准差参数相乘并除以对角线基础值后叠加到对应位置。此处需要实现随机Galerkin投影矩阵的构建逻辑。

第二部分功能：构建空间与随机场耦合的全局系统矩阵。
算法思路：需要分别获取空间离散矩阵的单元数和PCE的维度。调用第一部分的功能生成PCE矩阵。随后，分别构建空间域和随机域的单位矩阵，利用克罗内克积将空间单位矩阵与PCE矩阵相乘，再加上空间矩阵与随机域单位矩阵的克罗内克积，从而实现空间算子与随机算子的解耦叠加。

第三部分功能：蒙特卡洛模拟与PCE解析解的对比验证。
算法思路：生成指定数量的标准正态分布随机样本，结合均值和标准差参数得到随机场样本。此处需要实现指数衰减随机微分方程的精确解计算，即在给定终止时间下，计算初始标量值与随时间指数衰减项的乘积。基于样本计算蒙特卡洛的均值和方差。同时，需要计算PCE的解析均值（即零阶系数，根据高斯随机场指数的期望解析性质推导，涉及均值、标准差与时间的二次项组合）。最后，计算蒙特卡洛均值与解析均值的相对误差，并将蒙特卡洛均值、方差、解析均值、相对误差及样本数以字典形式返回。
